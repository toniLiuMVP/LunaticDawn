[ 開局與設定 ]

第一步：DOSBox 環境設定

未來之書是 1998 年 DOS / Win95 時代作品，現代作業系統需用 DOSBox 模擬。建議參考俠客遊 II 啟動器頁面的 DOSBox-X 配置，原理通用：

  • 設定 cycles = fixed 50000（Pentium 100 等效）
  • 掛載遊戲目錄為 C:\
  • 音效配置：SoundBlaster 16 + GM MIDI

第二步：角色設定

遊戲開始時要設定角色：

  • 姓名：可自由輸入（中文 1-3 字）
  • 稱號：可在遊戲中變更，影響 NPC 反應
  • 生日：跟隱藏角色密技密切相關（特定生日 + 特定姓名 = 觸發隱藏角色）
  • 性別：開局決定不能變
  • 家系：影響起始金錢 / 屬性
  • 職業：戰士 / 魔法師 / 盜賊 / 僧侶（影響技能起始值）

※ 想體驗「神級裝備」直接玩可以用隱藏角色密技，但建議先正常玩通關再試。

[ 跟俠客遊 II 的相似 / 差異 ]

沿襲俠客遊 II 的核心設計

  • 開放世界 RPG 結構（無強制主線，自由探索）
  • 城鎮 + 地下城 + 戰鬥地圖三段式
  • 旅店接委託 → 任務 → 賺錢 → 升級裝備循環
  • 同行 NPC 系統（招募有技能的角色一起冒險）
  • 5 系魔法（物理 / 元素 / 神聖 / 暗黑 / 精靈）

未來之書新增 / 改良

  • 新地圖區域（俠客遊 II 沒有的城鎮）
  • 新裝備類型（部分是俠客遊 II 沒見過的）
  • 更精緻的圖像（受惠於 1998 PC 性能進步）
  • 系統優化（戰鬥反應 / UI 操作）

※ 核心系統跟俠客遊 II 高度共用，玩過 II 的玩家上手未來之書會很快。

[ 第一週玩法建議 ]

Day 1-2：熟悉介面 + 接低風險委託

  • 到旅店看委託列表，選擇「送貨 / 護衛」這類低風險任務
  • 累積初步資金（避免一開始就接「擊退」「決鬥」這類高風險）
  • 到武器店看裝備價格，估算多少天能買到第一把好武器

Day 3-7：技能訓練 + 招募同行

  • 到訓練所提升技能熟練度（劍 / 棍棒 / 投擲等）
  • 到酒吧 / 公會招募有互補技能的同行 NPC
  • 5 系魔法開始學習：物理系 → 元素系 → 神聖系 / 暗黑系（看玩家路線）

Week 2+：進入地下城探險

  • 準備充足燈油（建議 50 小時以上）
  • 準備鑰匙（或招募有盜賊技能的同行替代）
  • 從低層地下城開始，避免一開始就挑戰高難度迷宮
