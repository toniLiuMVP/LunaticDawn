[ 觸發機制 ]

未來之書內建 5 個隱藏角色，透過角色設定畫面輸入特定姓名 + 生日 + 任意稱號即觸發。

設定步驟

  1. 遊戲開始進入「角色設定」
  2. 姓名：完全照下方表格輸入（含 ! / @ 等符號）
  3. 稱號：任意（寫「隨便」也可，5 人之一 M.WADA 限定 MASTER）
  4. 生日：按表格指定的月 / 日（其他選項可隨意）
  5. 確認後遊戲會觸發隱藏角色屬性 + 裝備

※ 後 2 人（Caspia / K.OOHARA）一開始就有家（位於「群星之鎮」），不需到處找住處。

[ M.YOSHIDA! · 富豪型 ]

  • 姓名：M.YOSHIDA!（最後是驚嘆號）
  • 生日：11/30
  • 稱號：隨便
  • 特性：起始錢很多

適合「想跳過初期練功階段」「直接買裝備衝劇情」的玩家。

[ Hydrogen!! · 神級屬性 ]

  • 姓名：Hydrogen!!（兩個驚嘆號）
  • 生日：8/10
  • 稱號：隨便
  • 特性：永遠不老 + HP/MP 1998

「神的能力」，不會因為遊戲內時間流逝而老死，HP/MP 各 1998（一般角色 100-300 範圍）。

※ 1998 是遊戲發行年份的彩蛋。

[ M.WADA · 惡魔裝甲 ]

  • 姓名：M.WADA
  • 生日：12/14
  • 稱號：MASTER（必填，不能寫其他）
  • 特性：全身的惡魔裝甲

整套惡魔裝甲一開始就配發，防禦力高。但屬性沒有特別誇張。

[ Caspia@@@ · 第一強類 ]

  • 姓名：Caspia@@@（三個 at 符號）
  • 生日：8/24
  • 稱號：隨便
  • 特性：很強 · 一開始就有家（群星之鎮）

[ K.OOHARA! · 終極神裝 ]

  • 姓名：K.OOHARA!（最後是驚嘆號）
  • 生日：7/22
  • 稱號：隨便
  • 特性：蠻厲害的 · 一開始就有家（群星之鎮）

「大原系」裝備分析

K.OOHARA 取自開發者大原（Oohara）的姓氏，相關裝備強度爆表：

  • 所有屬性全部上升
  • 攻擊力無與倫比：普通敵人砍 1 刀掉 999 HP（單發秒殺）
  • 防禦力幾乎無敵：大頭目攻擊只少 1-15 HP
  • 整體強度大概是「除錯模式 / 開發者模式」級別

※ 用 K.OOHARA 玩等於開「無敵模式」，適合單純想「看劇情 / 無壓力探索 / 體驗所有地圖」的玩家。

[ 推薦使用時機 ]

  • 第一輪：用一般角色正常玩，體驗成長感
  • 挑戰失敗想試其他玩法：用 M.YOSHIDA! 或 Hydrogen!!（中等強度）
  • 純粹想看劇情 / 無壓力遊覽：直接 K.OOHARA!（最強）
  • 不建議：第一次玩就用 K.OOHARA，會破壞遊戲樂趣
