●俠客遊.未來之書: 超神一族

  這是小弟應用前面兩篇資料的修改範例，因為是實測過，所以
把實際的資料附上，讓同好們方便比對。

●準備工作

1.創造世界：保持平均（好像剛灌好遊戲不要改變設定，直接創
  造新世界的結果都會一樣，連每個迷宮所在的城市都可以用小
  弟先前畫的那個地圖）

2.選擇人物：10∼20歲，男魔法師，中津國(有灣岸都市那個，剛
  開始送貨近)
	→彬彬有禮的 郭 勺光
  輸入名字、外號、出生日期
	→jesse、超美形的、2月20日(為了妖精之國)
  選擇人物造型：...

3.調好你的視窗，設定遊戲速度，脫下全部裝備，去旅舍睡個14
  天後存檔

4.找到人物位置 (我常用Uedit或FPE)
％建議位置(位置為16進位)
 00: 編號
 08: 姓名			<-- 出現次數太多，不好找
 11: 外號			<-- Uedit找這個比較快
 e0: 錢				<-- 很好找，用8 byte更快
14c: HP, MP, MaxHP, MaxMP	<-- FPE找這個比較快，用個恩惠符就好了

1d4是第一個物品的編號，下4 byte是名稱，每個物品長度d8

●●●● 修改範例(Ⅰ) ●●●●
jesse曰：
－最好連名稱一起改，否則會不小心賣掉
－記得照下面順序先輸入low byte，再輸入high byte

▲測敵 木棍：就選一開始就有的那個
  攻擊範圍	48		= 06
  命中		50		= 64
  攻擊力	58		= e7 03 (有死亡玉的殺傷力，帥吧？)
  使用效果	b4		= 0c 00 00 00 (可測敵，原為ff ff ff ff)

▲尋寶 紙甲：就選一開始就有的那個
  防禦力	5c		= e7 03 (不要大意, 挨一下還是會HP -1)
  魔法屬性	68, 6c,..., 84	= 5a, 5a,..., 5a (連續兩行)
  屬性提升	88, 8c, 90, 9c	= 5a, 5a, 5a, 5a (連續一行)
  使用效果	b4		= 0d 00 00 00 (可尋寶，原為ff ff ff ff)
  屬性加成效果	c4, c8, cc, d0	= 0a, 0a, 0a, 0a

▲好用的 復力丹: 一開始有一個
  使用效果	b0, b4, b8, bc	= 09, 09, 09, 09 （原06, 改為滿力丹）
  用後消失機率	c0		= 00  （用不完）

％直接買個滿力丹，就只要改c0，比較好用(因為有時候MP不足，但電腦不會自
  動以復力丹幫隊友補MP)

▲不熄的 燈籠：一開始有一個
  用後消失機率	c0		= 00  （用不壞）

％最後記得把超神裝備戴上去

jesse曰
：開始慢慢的送貨吧。什麼？連＄都一起改了？第一次玩的人不要這樣啦...


●●●● 修改範例(Ⅱ) ●●●●

迷宮走累了嗎？同伴死掉了嗎？到寺院買個恩惠符和安寧符，組隊逛迷宮時，
下面兩個東東很有用

▲脫離 恩惠符: 初期賣出價$357
  使用效果	b0, b4, b8, bc	= 2d, 2d, 00, 00 （原3a, 改成脫離水晶）
  知性限制	a0		= 0a （知性不滿30的要改這個）

▲轉生 安寧符: 初期賣出價$964
  使用效果	b0, b4, b8, bc	= 00, 3d, 3d, 3d （原38, 改成轉生符）
  知性限制	a0		= 0a （知性不滿40的要改這個）

▲永恆的 返老丸：如果你國慶日摸彩摸到頭獎
  用後消失機率	c0		= 00  （老不了...）

jesse曰
－危險的時候再把測敵木棍拿再手上，平時空手打再練練魔法就好（經驗值看出
  手次數，而非出手結果），不然同伴賺不到什麼經驗值，久了會怨恨你的
－這種裝備還是要注意隱形陷阱，有些一次損血數十點，該開始的人物來不及用
  好用的 復力丹，就呵屁了...
