: :   因為我隊友中有一個弓箭手，結果偷到一些XX弩(加敏捷)、暴X箭、連發x槍、
: : 左輪x槍...，轉交給他之後都不用，不過暫時沒賣掉就是了。這種情形好像每
: : 個人都一樣，像我把妖刀村正給一開始罩我的老揹揹（我喜歡用圓月彎刀配聖
: : 盾，全身合計的敏捷度會有 115），把聖盾和一把xx(長柄)槍給我耍槍的老婆
: : （柔，是我上一版老婆的名字。這次好多人名和上次一樣），結果都是沒用，
: : 但較好的裝備還是沒賣，反而佔了他的空間...

超強，我一直以為武者寶珠是最強的，現在終於明白魔法的威力更強。謝謝！  :>

  不過，最後還是把落雷水晶放回家了。因為多偷幾次之後，屬性下降，威力就變
差了，守成不易啊。目前全部用妖精國的物品來強化魔法，然後身上帶著死亡玉，
各種巨人、九頭蛇，甚至連各種龍，都只要一次就ＯＫ了，強的沒話說，雖然對大
