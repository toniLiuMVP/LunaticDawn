📜 戰鬥系統概要

俠客遊 II 戰鬥系統是 1996 年同類 RPG 中複雜度最高的之一，由 6 大子系統組成：

  1. 8 大屬性（力 / 智 / 靈 / 敏 / 直 / 魅 / 體 / 精）— 影響所有 stat checks
  2. 15 技能熟練度（劍 / 槍 / 棍 / 弓 / 投 / 拳 + 5 系魔法 + 治癒 / 探查 / 偷 / 盜）
  3. 5 系魔法（物理 / 元素 / 神聖 / 暗黑 / 精靈）
  4. 8 武器類必殺技（拔刀波斬流 / 暴棍撲術 / 等）
  5. 7 弱點屬性（野獸 / 魔獸 / 魔法生物 / 巨人 / 不死 / 幽靈 / 沒有）
  6. 7 攻擊方式（普 / 砍 / 飲 / 散 / 跨 / 刺 / 敲）+ 對應 3 軸防禦

⚔️ 7 種攻擊方式

攻擊方式對應武器特性對應 3 軸防禦
普（普通）多數武器預設無特殊效果，平均傷害砍 / 刺 / 敲 平均
砍（斬擊）劍類 / 大刀 / 雙手刀類對裸甲傷害高防砍（byte 0x40）
刺（穿刺）長槍 / 槍類 / 短矛穿透重甲，對重裝敵特效防刺（byte 0x42）
敲（衝擊）棍棒類 / 鎚 / 鏈枷對骨骼類有額外效果防敲（byte 0x44）
飲（吸取）暗黑系武器 / 邪能傷害敵人同時恢復玩家 HP防（魔法）
散（散彈）多目標武器同時打多個敵人，單體 dmg 較低各個防禦
跨（跨距）弓類 / 投擲類遠距離攻擊，可隔行跨行各個防禦
戰術選擇：怪物 binary 中有「防砍 / 防刺 / 防敲」三軸（offset 0x40 / 0x42 / 0x44）— 判斷怪物類型選對應攻擊方式：
  • 骷髏 / 髑髏兵 → 敲系（棍棒對骨骼有效）
  • 重甲戰士 / 騎士 → 刺系（長槍穿甲）
  • 軟身怪物（蛇 / 蝙蝠）→ 砍系（劍直接斬殺）
  • 群體小怪 → 散 / 跨（弓 / 投擲一次多隻）

📊 戰鬥系統 binary 對照

玩家 / NPC 角色（LUNACHAR.SAV byte offsets）

0x36-0x3D: 8 大屬性 (1 byte each) — 力 / 智 / 靈 / 敏 / 直 / 魅 / 體 / 精 0x3E-0x45: 8 屬性上限 (1 byte each) 0x78-0x86: 15 技能熟練度 (1 byte each, 0x87 為空隙) 0x88: 物理系魔法 (u16 LE bitfield) 0x8A: 元素系魔法 (u16 LE bitfield) 0x8C: 神聖系魔法 (u16 LE bitfield) 0x8E: 暗黑系魔法 (u16 LE bitfield) 0x90: 精靈系魔法 (u16 LE bitfield) 0x92: 武技 (u16 LE，可裝備到 0xFFFF 全武技)

怪物（MONSTER.DAT byte offsets）

0x14: HP (u16 LE) 0x16: MP (u16 LE) 0x18: STM 體力 (u16 LE) 0x1A: 等級 Lv (u16 LE) 0x24: 經驗值 (u16 LE) 0x3E: ATK 攻擊力 (u16 LE) 0x40: 防砍 (u16 LE) 0x42: 防刺 (u16 LE) 0x44: 防敲 (u16 LE)

🎮 6 種戰鬥指令完整定義（說明書 p160）

指令機制
喊叫向部隊同伴打聲招呼指示行動，可向個別 or〔全體人員〕發令
解救讓部隊某人對另一人進行救援治療
治療藉魔法治療某人，只對個人，無法對全體
跳躍（撤退）戰鬥中讓某人撤退；向全員指示撤退時自己也要撤退
攻擊敵人攻擊指定人物；若指定〔全體人員〕則全部隊一起攻擊共同敵人
使用道具讓此人使用所擁有的道具（個人指定，不可全體）

💡 重要機制：地下城內若門上鎖，物理系魔法可以打開鎖（不必依賴鑰匙 / 盜賊技能 — 說明書 p140）。

🗺 戰鬥地圖 UI（說明書 p155）

  • 畫面右下方有縮小的戰鬥領域全部地圖（〔戰鬥地圖〕）
  • 四角框之中的部份 = 上方畫面當前顯示的部份
  • 戰鬥地圖外側的鋸齒狀物 = 牆壁
  • 戰鬥場所地形 = 從地下城地形而來（戰鬥場景連動地圖）

位置移動控制：拉動鈍色方框 → 移動戰鬥地圖；地圖上有 8 個方向窄圖，按一下即向該方向移動。

⚡ 戰鬥流程 & 行動順序

① 速度判定

敏捷屬性（byte 0x39）決定行動先後。敏捷高的角色先行動。

建議：練「敏捷」對所有戰鬥都重要 — 先攻佔便宜。

② 命中判定

武器命中率 + 玩家「直覺」（byte 0x3A）+ 武器技能熟練度 → 計算最終命中率。

建議：選一兩種武器類別練到熟練度 90+。

③ 傷害計算

武器攻擊力 + 屬性加成 - 敵人對應防禦（防砍 / 防刺 / 防敲）= 最終 dmg。

必殺技傷害 = power 值（HISSATSU.MST byte 4-5）— 跳過普通計算。

④ STM 消耗

每次攻擊消耗 STM（體力）。普通攻擊 ~5-10；必殺技 ~85-118（變體更多）。

STM 為 0 時無法攻擊，需休息回復。

⑤ MP 消耗

魔法施展消耗 MP。低階法術 ~10-30 MP；高階 ~50-100 MP。

MP 不足時換施低階魔法或補 MP。

⑥ 狀態異常

中毒 / 失明 / 麻痺 / 沉默 / 石化等。用對應道具 / 神聖系魔法解除。

boss 多用狀態異常 — 準備充足解除道具。

🛡️ 防禦戰術

① 物理防禦

  • 頭部護具：流星盔 / 羽飾盔 / 名甲 / 鐵甲（防砍 / 防敲為主）
  • 身體防具：鋼甲 / 鎧甲 / 厚皮甲 / 軟皮甲（防砍 / 防刺平衡）
  • 手部護具：鎧手套 / 護腕 / 鐵拳（防敲為主）
  • 腿腳護具：鏈甲靴 / 強盜靴 / 皮靴（防刺）
  • 盾類：方盾 / 圓盾 / 龍鱗盾 / 銳刃盾（全方位防禦）

② 魔法防禦

每件武器 / 防具都有「魔法防禦上限」（攻略書 PART 5 ch2 part2 第 7-12 欄）：

  • 聖 / 火 / 水 / 地 / 風 / 刀 / 力 / 神聖 — 8 種魔導屬性
  • 強化武器 / 防具到 32 級可大幅提升魔法防禦

③ 狀態異常防禦

  • 解毒石 — 解除中毒狀態
  • 解沈靜符 — 解除沉默狀態
  • 防盲目鏡片 — 防止失明
  • 守護娃娃 — 全狀態防禦

④ 神聖系守護法術

  • 祝福術 — 提高隊伍 STM
  • 護身術 — 提高指定角色防禦力
  • 封魔術 — 全隊免疫對方魔法

📈 戰鬥能力提升路徑

  1. 新手期：練 1-2 種武器類別到熟練度 30+，敏捷 / 力量提到 12+
  2. 初期：學物理系魔法（照明 / 開鎖 / 索敵 / 護身）— 探索 + 防禦兼顧
  3. 中期：選元素系（攻擊）或神聖系（治療）為主魔法路線
  4. 中後期：必殺技主形 + 強化武器到 16-24 級
  5. 後期：必殺技變體（STM ≥ 150）+ 強化武器 32 級 + 暗黑系狀態異常
  6. 終局：5 系魔法全會 + 4 召喚獸 + 神器級武器（火炎神斷 / 神切劍 / 火焰神杖）

🔬 戰鬥的隱藏細節

狀態異常持續性：中毒會每回合扣 HP；麻痺停止行動 N 回合；石化是「永久」需道具解。boss 用石化超強！

STM 回復速度：戰鬥外慢回；旅店 / 城堡可瞬間補滿。長戰前先補 STM。

必殺技冷卻：發動後不能立刻再發 — 切換普通攻擊幾次再發。

同行 NPC AI：自動戰鬥但會浪費 STM / MP — 用「指令」按鈕介入。
