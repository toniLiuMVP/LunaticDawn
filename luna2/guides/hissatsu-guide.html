📜 必殺技系統概要

俠客遊 II 的必殺技系統由 HISSATSU.MST（位於 遊戲資料夾/P/HISSATSU.MST）控制。 本站2026從 LUNA2.EXE 二進位提取出 8 個必殺技名稱字串（位於 EXE 偏移 0x03FCC2-0x03FD07）， 2026 / 2026 / 2026完整逆向 32 records 結構。

必殺技 vs 一般攻擊：必殺技是「特定條件下發動」的強力攻擊（高 STM 消耗 + 動畫專用）， 比一般攻擊威力高 2-4 倍。每種武器類別只能用對應系列的必殺技。

📊 STM 消耗 vs 威力對照

power / STM_cost 比率（投資效率）： - 拔刀主形：300/90 = 3.33× - 拔刀變體：450/110 = 4.09×（更高效率） - 暴棍主形：310/85 = 3.65× - 暴棍變體：460/115 = 4.00× - 真槍變體：490/118 = 4.15×（最高效率） - 波動主形：200/60 = 3.33×（內功系最低 STM 消耗） - 斷強主形：250/70 = 3.57×（防禦系，特殊效果）

建議： 變體形態普遍比主形效率高 4-15%，但 STM 消耗也高 20-30。 玩家初期 STM 不高時優先用主形；STM 練上去後切換變體。

STM 消耗 pattern：

  • 12 個 common（id 16-21, 26-31）：b1=60，威力 250-500，無動畫（b2-3 = 0xFFFF）
  • 4 個 low_cost（id 22-25 = 4 元素精靈）：b1=15 或 10，威力 100，元素精靈快攻特性

怪物使用這些必殺技時，怪物資料庫的「必殺技 O/X」欄位會標 O。 本站必殺技資料庫有 32 records 完整列表 + parser 結構。

🎯 玩家路線建議

  1. 新手期：用主形必殺技（85-92 STM 消耗），威力 300-340 階段
  2. 中期：練 STM 體格到 150+，切換變體形態（威力 400-490）
  3. 高 STM 角色：可頻繁發動變體（4-5× 投資效率）
  4. 低 STM 角色：選波動六骸法（內功系，主形只 60 STM 消耗）
  5. 對 boss：用變體 + 弱點屬性武器（請見弱點與加成攻略）

🔬 技術細節（給逆向有興趣的人）

HISSATSU.MST 結構（每 record 36 bytes）：

byte 0: record id (0-31) byte 1: 變體碼 / 推測 STM 消耗（待遊戲內實測） byte 2-3: u16 LE 動畫 sprite ID（records 16-31 = 0xFFFF 無動畫） byte 4-5: u16 LE power（威力值） byte 6-7: u16 LE flags（per-技 multi-bit flag） byte 8-9: u16 LE secondary power（暴擊 / 連擊 power） byte 10-15: 細部設定（specials 用作冷卻時間 / AOE 範圍） byte 16: ★ 怪物 ID lookup（records 16-31 對應 MONSTER.DAT index） byte 17-35: 19 bytes 細部設定 + padding

parser 程式：tools/extract/ 內。
