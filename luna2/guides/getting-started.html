📜 歡迎進入《俠客遊 II》的世界

1996 年 Artdink 製作的《俠客遊 II》（Lunatic Dawn II）是同年代最複雜的開放世界 RPG 之一。 玩家在 6 個國家自由旅行，接任務 / 練武 / 學魔法 / 招募 NPC / 挑戰魔王。 本指南幫助新手在第一週上手，避開常見地雷。

建議閱讀順序：本指南 → DOS 執行 FAQ（解決執行問題）→ 武器系統 + 玩家成長路徑（理解戰鬥）→ 其他攻略。

⚡ 8 大基礎屬性

每個角色（玩家 + NPC）都有 8 大屬性，從攻略書 PART 1 ch1 + LUNA2.EXE 偏移 0x0401A8 提取確認：

力量（力）

物理攻擊力的基礎，決定劍 / 斧 / 棍 / 槍系武器威力。

劍 = (力×2 + 敏)÷3

智力（智）

魔法系統的智力強度，影響魔法學習速度 + 成功率。

學魔法所需智力門檻

靈巧（靈）

手部精細動作 — 開鎖 / 解陷阱 / 偷竊技能基礎。

盜賊系技能基礎

敏捷（敏）

行動速度 + 命中率輔助。先攻 / 閃避取決於敏捷。

行動順序 / 命中率

直覺（直）

直覺反應 — 弓 / 投擲類武器命中 + 偵測陷阱。

弓 / 投擲 = (力+敏+直)÷3

魅力（魅）

招募 NPC 成功率 + 商店買賣價差。

影響招募 / 貿易

體格（體）

HP / STM 上限基礎，決定武器搬運能力。

槍 = (力+體×3)÷4

精神（精）

MP 上限 + 抗魔法能力 + 神聖系威力基礎。

MP 上限 / 抗魔

📈 屬性訓練機制（說明書 p025）

  • 每項能力上限 = 100（hard cap，跟 LD 存檔修改器嚴格模式 A 對齊）
  • 角色起始能力決定訓練可達上限：起始很低的能力（例：力量 20）訓練能提升有限
  • 均衡設定可達 100：「只要好好的計算一下，還是有可能將魅力提升到最高值的一百」
  • 新手建議：角色設定時不要把屬性壓到極低（會卡死訓練上限）— 均衡分配優於偏重

🧝 6 種族（含半精靈人）

種族特色
人類主流種族，平衡屬性
精靈壽命長（約人類兩倍長），魅力 / 智慧偏高
半精靈人獨立種族（不是 1/4 人類）— 擁有人類 + 精靈各一半的特性，數量比精靈少

獸人野性 / 體力

💡 半精靈人的歷史：人與精靈的祖先原本相同，演化才形成人類、精靈、半精靈人三種獨立種族（說明書 p035）。

👤 8 種出身選擇

遊戲開始時玩家從攻略書 PART 1 提到的多種出身選擇（部分如下）：

① 貴族

所有屬性平均高 ~141；起始金錢 10000 G；魅力為主。建議玩家：英雄之路 / 金錢之路。

② 農家

力量 / 體格高 156；起始金錢 89 G；劍 / 斧 / 槍適合。建議玩家：英雄之路（戰士類）。

③ 戰士

力量 / 體格 / STM 高；劍 / 雙手刀類武器。建議玩家：英雄 / 邪惡之路。

④ 武士

力量高 + 武術技能高；雙手刀 / 太刀類。建議玩家：英雄之路。

⑤ 騎士

力量 + 體格 + 神聖魔法；長槍 / 騎兵武器。建議玩家：英雄之路（神聖魔法路）。

⑥ 魔術師

智力 / 精神高；元素 / 物理魔法精通。建議玩家：邪惡 / 金錢之路（暗黑魔法）。

⑦ 祭司

智力 / 精神 / 神聖魔法熟練度高。建議玩家：英雄之路（神聖路線）。

⑧ 盜賊

靈巧 / 敏捷高；偷竊 / 解鎖 / 陷阱偵測。建議玩家：邪惡 / 金錢之路。

📐 HP / MP / STM 計算公式（攻略書 PART 1 ch1 p017）

能力公式意義
HP（生命力）((力量 × 4) + (體格 × 4) + ...) ÷ 4降為 0 = 死亡
MP（魔法力）((智力 × 4) + (直覺 × 4) + (精神 × 3)) ÷ 3魔法施展能量
STM（耐力）((力量 × 2) + (體格 × 4) + (精神 × 5)) ÷ 4移動 / 戰鬥消耗，0 時 HP 開始下降

📅 新手第一週玩法

1

第 1-2 天：神聖吉札帝國 歐德普克

  • 進入主城歐德普克（首府）
  • 逛 5 大店家：武器店 / 道具店 / 魔法屋 / 訓練所 / 旅店
  • 練 1 種武器類別到熟練度 5（每級 250 G，5 級花 1250 G）
  • 到神聖系寺院學 1 個基礎神聖魔法（治療術）
2

第 3-4 天：第一個任務

  • 到旅店接任務（送貨 / 送信類，難度 Easy）
  • 報酬 500-2000 G — 第一筆「主動」收入
  • 跟著驛馬車到目的地 → 完成 → 回報酬
  • 完成 2-3 個任務後 → 等級提升 + 名聲累積
3

第 5-6 天：學物理系魔法

  • 到魔法屋學 3 個必備物理系魔法：照明術（地下城用）+ 索敵術（顯示敵人）+ 開鎖術
  • 這 3 個對地下城探險絕對必備
4

第 7 天：招募第一個 NPC

  • 到旅店 / 城堡周邊找適合 NPC
  • 選別號「戰士」「魔法師」類，種族「人類」
  • 魅力 ≥ 12 提高成功率
  • 邀請後組隊 → 戰力翻倍
  • 看 同行 NPC 招募流程 詳細
5

第 2 週起：跨國

  • 準備跨國通行證（神聖吉札 → 敦津 / 樸哈爾）
  • 探索新國家 — 解鎖訓練 / 魔法 / 武器
  • 選定主要路線（英雄 / 邪惡 / 金錢）
  • 看 武器系統 + 玩家成長路徑 規劃

🔐 1996 年校驗碼機制（玩家須知）

俠客遊 II 是 1996 年第三波代理的 DOS RPG，含當年的防盜版設計：

  • 機制：遊戲在某些劇情點會跳出校驗碼查驗對話框，問「請輸入第 N 號角色的校驗碼」
  • 解法：原版說明書最後印有 24 個角色頭像 + 對應 8 位數字，玩家依 N 號查找輸入
  • 遺失說明書怎麼辦：建議至 Steam 購買正版（Artdink 專區）+ 取得說明書 PDF；或在 toni 測試的懶人包（Luna2_EZ）內附參考

💡 歷史趣味：1990s DOS 遊戲普遍有此設計（玩家無說明書 → 無法繼續遊戲）— 反映當時對抗盜版的時代背景。本站基於數位文化保存目的，不公開列校驗碼數字本身（請玩家透過正規管道取得說明書）。

⚠️ 新手地雷

① 沒 save 就死了：俠客遊 II 沒自動 save。死亡 = 重最後一次 save。 務必每進新區 / 接任務後 save。USER0-USER9 共 10 個 slots — 多用幾個。
② 一開始全屬性練：分散 ROI 低。 選 1-2 個主屬性（按出身 / 路線）+ 1-2 種武器類別專練。
③ 武器太重不能裝備：每件武器有「體格需求」（攻略書 PART 5 ch2「體格」欄）。 力量 / 體格不夠拿不起來。**買武器前看體格需求**。
④ 跨國沒通行證被擋：不同國家間移動需要對應「城（通行證）」（如敦津 25000 G / 馬尼卡帕王國 25000 G）。 跨國前確認金錢 + 通行證。
⑤ 訓練選錯路線：同武器類在不同國家可選不同訓練深度。 江都（敦津）訓練所最完整 — 含元素系魔法。建議中期玩家去那。

🎮 操作技巧（DOSBox）

俠客遊 II 是 DOS 遊戲，需用 DOSBox 模擬器執行（本站跨平台啟動器已封裝）：

  • 速度調整：cycles=fixed 50000（Pentium 100 等效，符合 1996 設計）
  • Save/Load：F2 選擇存檔槽 → S 儲存 / L 讀取
  • 地圖移動：方向鍵走動，Enter 對話 / 互動
  • 戰鬥指令：A 攻擊 / M 魔法 / S 必殺技 / D 防禦 / R 逃跑
  • 退出 DOSBox：Ctrl+F9 或 exit 命令

詳細執行問題請看 DOS 執行 FAQ（青衫 1990s 末撰寫，至今適用）。
