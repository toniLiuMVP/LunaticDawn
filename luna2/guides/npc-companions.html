📜 NPC 系統概要

俠客遊 II 是開放世界 RPG，玩家可以在各地招募 同行 NPC 組成隊伍。 NPC 系統分三層：

  1. NPC 字典（660 個獨特字串）— 遊戲生成 NPC 名字的字典池
    • CALLNAME 別號 180 + MCNAME0 男名 192 + MCNAME1 姓 192 + FCNAME 女名 96
    • 男性全名組合空間 36,864 種；含別號約 6.6M 種
  2. 重要 NPC 數值表（攻略書 PART 5 ch1，150-170 個）— 含 8 屬性 + 15 技能熟練度
    • 對應存檔修改器 byte 0x36-0x86 結構
  3. 玩家親分 / 子分系統（KOBUN.SAV / OYABUN.SAV）— 玩家當「老大」帶「小弟」
    • 2026 binary 逆向初步釐清結構

🤝 NPC 招募流程

1

遇到 NPC

玩家在城鎮 / 旅店 / 任務地點遇到 NPC。每個 NPC 有：別號 + 名 + 姓組合（隨機從字典生成）+ 屬性數值（攻略書記錄重要 NPC）。

2

觀察 NPC

判斷 NPC 種族 / 職業 / 別號是否符合需求。看別號很重要：「戰士」「武術高手」「魔法師」清楚標明強項；「商人」「吟遊詩人」適合金錢路線。

3

邀請加入隊伍

跟 NPC 對話選擇「邀請」選項。NPC 會根據自己的等級 / 個性 / 玩家魅力決定是否加入。 魅力屬性高的玩家招募成功率高（玩家屬性 byte 0x3B）。

4

建立子分（小弟）關係

NPC 同意後加入隊伍，記錄到 USER0/KOBUN.SAV（542 bytes）。 2026 binary 結構釐清：含小弟 ID + 各種數值（u16 LE 多欄位 — 13/16/32 等可能是「忠誠度 / 等級 / 屬性」）。

5

玩家成為親分（老大）

當隊伍人數達到一定，玩家本身會記錄到 USER0/OYABUN.SAV（258 bytes）。 2026結構初步釐清：byte 0-7 = 玩家親分編號 / 隊伍人數；byte 8+ = 屬性概要（u16 sequence 971/7/12/10/30 等）。

6

同行戰鬥 / 探險

同行 NPC 自動戰鬥（按 AI），玩家可看狀態 + 給予命令。NPC 可裝備武器（從 CHRITEM.SAV 對應 char_id 物品欄）。

7

NPC 流失條件

NPC 可能因下列原因離開：

  • 戰死（HP 歸零未復活）
  • 跟玩家路線不合（暗黑路線時神聖系 NPC 會離開）
  • 玩家攻擊同伴（魅力降低 + NPC 反目）
  • 長期不付薪水（少數金錢路線 NPC）

🗝️ 盜賊技能的關鍵價值

說明書 p085 + 攻略書 PART 3 ch2 多次強調盜賊技能的不可替代性：

盜賊能力替代方案差異
解除地下城門鎖萬能鑰匙（耗材，每支用一次就掉）盜賊可重複使用，省鑰匙費
解除寶箱機關無替代盜賊獨有 — 沒盜賊則跳過寶箱
偷取 NPC 物品無替代盜賊獨有特殊技能
地下城特殊機關陷阱需熟練度高的角色 + 神秘智者 NPC盜賊解除效率最高

💡 說明書原文要點：「若是隊伍裡有精通盜賊技能的人，那麼就算是沒有萬能鑰匙也沒什麼關係的」+「招募擁有優秀盜賊技能的夥伴，待遇從優」（說明書 p085 印頁 P78）。

⚠️ 另類解法：地下城內若門上鎖，物理系魔法也可以打開鎖（說明書 p140 印頁 P132）— 不必依賴鑰匙 / 盜賊。但盜賊解寶箱仍然不可替代。

🎯 招募策略建議

① 早期招募（Lv 1-15）

  • 目標 NPC 類型：「戰士」「武術高手」「物理魔法高手」
  • 主要城鎮：神聖吉札帝國 歐德普克 / 路特卡得（旅店 / 城堡周邊）
  • 魅力建議：玩家魅力 ≥ 12 提高招募成功率

② 中期招募（Lv 15-30）

  • 目標 NPC 類型：「神聖魔法高手」「元素魔法高手」「賢者」「神之化身」
  • 主要城鎮：敦津江都 / 樸哈爾阿薩拉姆 / 日傑紫陽（神聖系寺院附近）

③ 後期招募（Lv 30-50）

  • 目標 NPC 類型：「勇敢女神」「太陽女」「薔薇女」「精靈魔法高手」
  • 主要城鎮：馬尼卡帕王國 聖米哥 / 埃爾門德 / 里歐庫列歐
  • 稀有種族：精靈 / 矮人 NPC 多在後期區出現

④ 暗黑路線招募

  • 目標 NPC 類型：「黑暗帝王」「黑暗使者」「黑暗騎士」「黑暗魔道師」「殺人高手」
  • 主要城鎮：暗黑系寺院（神聖吉札 / 樸哈爾 / 馬尼卡帕）

🔬 NPC 屬性對應 binary 結構

NPC 在 binary 中的數值結構（已逆向）：

  • NPC 屬性：byte 0x36-0x3D（8 大屬性 力/智/靈/敏/直/魅/體/精）+ 0x3E-0x45（屬性上限）
  • NPC 技能熟練度：byte 0x78-0x86（15 技能：劍/槍/棍/弓/投/拳 + 5 系魔法 + 治癒/探查/偷/盜）
  • NPC 物品欄：CHRITEM.SAV 對應 char_id × 768 bytes（16 物品 slots × 48 bytes）
  • NPC 隊伍狀態：KOBUN.SAV / OYABUN.SAV（2026部分釐清）

玩家可用存檔修改器修改自己的屬性 / 技能 / 5 系魔法；用物品欄編輯器 v0.2修改物品。 但 KOBUN / OYABUN edit 模式仍待擴充（2026 結構釐清，目前僅 viewer）。

🎯 玩家路線 + NPC 招募策略整合
