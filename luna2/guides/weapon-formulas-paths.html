📜 為什麼需要這篇

俠客遊 II 是開放世界 RPG，玩家初期最常困惑的是： 「我練什麼武器？」「我走哪條成長路線？」「哪個國家適合我？」 這篇攻略把攻略書 PART 1（武器系統公式） + PART 2（6 國地理深度版） + PART 4（三條玩家路線）整合， 讓玩家在新手期就能規劃整個 game 的成長路徑。

本篇是6 國地理速查（2026）的深度版， 補充武器系統 + 三條玩家路線 + 11 城鎮設施分類。

⚔️ 武器類別 vs 屬性對應公式

俠客遊 II 的武器系統有 6 大武器類別，每類的「相關技能」由「基礎技能」（8 大屬性）線性組合產生（攻略書 PART 1 ch1）：

① 劍類

劍 = (力量 × 2 + 敏捷) ÷ 3
主要看力量（×2 權重），輔助敏捷。寶劍刀類 / 雙手刀類武器發動。

② 斧類

斧 = (力量 + 體格) ÷ 2
力量 + 體格（重型攻擊）。命中率較低，攻擊力高。

③ 棍棒類（榴棒）

榴棒 = (力量 × 3 + 體格 × 2) ÷ 5
力量 × 3 為主，體格 × 2 輔，最重視蠻力。

④ 槍類

槍 = (力量 + 體格 × 3) ÷ 4
體格 × 3 主導 — 唯一不同的是槍要兩手使用。

⑤ 弓類

弓 = (基本力 + 敏捷 + 直覺) ÷ 3
三屬性平均（力量 / 敏捷 / 直覺）。遠距武器，缺技能威力降低。

⑥ 投擲類

投擲 = (力量 + 敏捷 + 直覺) ÷ 3
同弓類算法但投擲類武器（飛鏢 / 飛矢 / 投石）。

🛤️ 三條玩家路線（PART 4 ch1/ch2/ch3）

① 英雄之路（PART 4 ch1）

目標：成為傳說中的英雄，挑戰魔王群並拯救世界。

典型路徑：

  1. 新手期 神聖吉札帝國 歐德普克 → 學基礎武器 + 物理魔法
  2. 初期擴展 路特卡得 → 學進階神聖系
  3. 中期跨國 敦津江都 → 學元素系
  4. 後期 馬尼卡帕王國 聖米哥 → 學高階神聖 + 召喚獸
  5. 終局 芙庫基歐 → 挑戰魔王群（提斯卡托 HP 999）

建議職業：戰士 + 神聖魔法（騎士類）、貴族出身（攻略書 PART 1 提到）

② 邪惡之路（PART 4 ch2）

目標：走暗黑系成為邪神化身，使用暗黑系魔法 + 即死技。

典型路徑：

  1. 初期 神聖吉札帝國 → 學基礎物理
  2. 中期 樸哈爾帝國 阿薩拉姆 → 學暗黑系第一階（毒氣 / 失明 / 麻痺）
  3. 後期 馬尼卡帕王國 埃爾門德 / 里歐庫列歐 → 學高階暗黑系
  4. 學「震亡術」即死技 + 神器級武器（火焰神杖 / 制刀封套）

建議職業：魔術師 + 暗黑魔法、農家出身（PART 1 提到的另一基礎人物）

③ 金錢之路（PART 4 ch3）

目標：透過商業 / 賞金 / 各國特產品貿易賺錢。

典型路徑：

  1. 各國跨國貿易：樸哈爾帝國（香料 / 絨毯）↔ 日傑（妖刀 / 松茸）↔ 馬尼卡帕王國（蜂王乳 / 水晶）
  2. 賞金獵人路線（CALLNAME 別號 #98 賞金獵人 / #99 屠龍者 / #100 屠熊者）
  3. 商業活動：道具店「特產品」價差套利

建議職業：商人（CALLNAME #11）+ 高直覺 / 高魅力屬性

🎯 玩家路線總建議

  1. 新手選擇：根據出身（貴族 / 農家）+ 屬性傾向選 1-2 種武器類別練 — 不要全練（資源分散）
  2. 武器升級：每國的訓練所提供「劍 / 弓 / 棍棒 / 槍 / 投擲 / 拳 / 物理魔法」訓練 — 同一武器類別可在多國訓練深化
  3. 路線選擇：英雄之路（戰士+神聖魔法）vs 邪惡之路（魔法+暗黑系）vs 金錢之路（商人+貿易）— 三條路線可混合
  4. 神器追蹤：火炎神斷 / 神切劍 / 火焰神杖 是全遊戲最強裝備，需玩家完成特定任務取得
  5. 終局準備：Lv 45+ 學齊 5 系魔法 + 4 召喚獸（伊弗利朵 / 琴恩 / 克拉肯 / 貝希摩斯）
