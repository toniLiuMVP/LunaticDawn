📜 5 系魔法系統概要

俠客遊 II 的魔法系統由 MGCDATA.MST（位於 遊戲資料夾/P/MGCDATA.MST）控制， 共 80 records，本站於 2026 年完整逆向確認結構：

  • 5 系分區：每系 16 個 slot（0-15 / 16-31 / 32-47 / 48-63 / 64-79）
  • 有效 slot 數：46-50 個 PC 學習法術
  • 4 召喚獸：精靈系後段（伊弗利朵 / 琴恩 / 克拉肯 / 貝希摩斯）
  • 5 系名稱從 LUNA2.EXE 偏移 0x03FD10 起提取確認
學魔法的場所： 神聖系 → 神聖系寺院；暗黑系 → 暗黑系寺院；精靈系 → 精靈術屋；物理 / 元素系 → 訓練所或魔法屋。 詳細地理請見 6 國地理速查。

🪐 5 系完整列表

① 物理系魔法（10 法術）

攻略書 PNG p133。MGCDATA records 0-15（12 有效 + 4 空）。

  • 照明術在地下城中代替油燈照亮四周
  • 開鎖術將鎖前的門 / 箱打開
  • 擾鬼術讓被指定的人物擾動
  • 沉默術讓被指定的人物無法施展魔法
  • 索敵術顯示出敵人的位置
  • 停時術讓被指定的人物無法活動
  • 浮遊術讓被指定的人物浮在空中
  • 尋寶術顯示出當前迷宮層的寶物位置
  • 護身術讓被指定一名人物擁有較高的防禦力
  • 封魔術讓所有對人物所使用的魔法全部失效

② 元素系魔法（8 法術）

攻略書 PNG p134。MGCDATA records 16-31（10 有效 + 6 空）。

  • 爆炸術在指定人物身上發生爆炸
  • 波彈術指定範圍中無數波彈落下
  • 魔彈術讓被指定的人物受到能量攻擊
  • 暴風雨在指定地點產生龍捲風
  • 刺地術在指定地點產生混亂的能源
  • 衝擊術巨大的能源體面向地點以實質形態衝擊
  • 隕石術在指定地點引起大爆炸
  • 宇宙射段指定人物上方發生巨大能量柱

③ 神聖系魔法（11 法術）

攻略書 PNG p135。MGCDATA records 32-47（12 有效 + 4 空）。

  • 祝福術提高我方全部隊員的耐力 STM
  • 治療術恢復生命力，可至完全恢復
  • 法力迴復將被指定人物所擁有的法力借予使用
  • 治療盲目治療盲目狀況
  • 解毒術治療中毒狀況
  • 瓶解術讓昏睡中醒過來
  • 治療麻痺治療麻痺狀況
  • 消除魔法解除對指定人物所施展的一切魔法
  • 除魔大法清滅不死生物 / 幽靈等敵人
  • 沉睡術對一指定地點施法，使該地點的人物沉睡
  • 大療法恢復所有狀況到最高，治療死亡以外的各種狀況

④ 暗黑系魔法（9 法術）

攻略書 PNG p136。MGCDATA records 48-63（12 有效 + 4 空）。

  • 毒氣術讓被指定人物中毒
  • 失明術讓被指定人物失明
  • 麻痺術讓被指定人物麻痺
  • 奪能術奪取被指定人物的魔法能力
  • 酸雲術在指定地點會出現一片酸雲
  • 召魔術召喚出魔魘鬥，若是魔法遭到解除
  • 雲蜜術在指定地點產生一片雲，使所有受到影響的人麻痺
  • 石化術讓被指定人物石化
  • 震亡術對指定目標投出咒術，讓目標立刻死亡

⑤ 精靈系魔法（8 自然 + 4 召喚獸）

攻略書 PNG p137。MGCDATA records 64-79（8 自然 + 4 召喚獸）。

  • 鬼火術在虛幻中造出鬼火，向指定目標攻擊
  • 飛襲術使被指定的敵人無法行動
  • 石頭術指定地點落下許多石頭
  • 誘惑術讓被指定目標神志不清
  • 大海嘯在指定四週颳起大浪
  • 巨喚術召喚傳說中的女武神
  • 冰凍術讓被指定目標遭到冷凍
  • 夢魔術讓被指定目標睏睡

🔥 4 召喚獸（精靈系後段）

  • 伊弗利朵 (Ifrit)：火焰元素王
  • 琴恩 (Djinn)：風 / 智慧元素
  • 克拉肯 (Kraken)：水 / 海洋元素
  • 貝希摩斯 (Behemoth)：地 / 巨獸元素

召喚獸是精靈系最高階法術，需要對應元素的特殊條件才能學會。

📊 5 系比對表

💡 玩家路線建議

  1. 新手期：學物理系（照明 + 開鎖 + 索敵）— 探索必備
  2. 戰鬥輸出：選元素系（魔彈術 / 隕石術）或精靈系（鬼火 / 冰凍）
  3. 輔助 / 治療：神聖系必學（治療 + 法力迴復 + 大療法）
  4. 狀態 debuff：暗黑系（毒氣 / 麻痺 / 石化）— 對 boss 有奇效
  5. 終極：精靈系召喚獸（4 隻最強元素王）

🔬 技術細節（MGCDATA.MST 結構）

每 record 32 bytes，有效欄位（2026 逆向）：

  • system：5 系分類（物理 / 元素 / 神聖 / 暗黑 / 精靈）
  • mp_cost：MP 消耗
  • atk：攻擊力（元素系最高）
  • type：分類（治療 / 攻擊 / 偵測）
  • anim_id：動畫 sprite ID
  • is_empty_slot：空 slot 標記（每系 16 slot 中部分未使用）
