📜 為什麼需要這篇

俠客遊 II 是開放世界 RPG，玩家可以自由在 6 個國家之間旅行。 每個國家有 2-5 座城鎮，每座城有不同的設施組合。 新玩家最常問的問題是：「我要在哪裡買武器、學魔法、找訓練、坐船？」這篇攻略整理出 6 國的結構性速查。

本篇規模：本篇只列國家 / 城鎮 / 設施類型 / 驛馬車路線結構。
具體物價、武器名稱、特產品清單請查 1996 年原版攻略書 PART 2「地理情報」，本站不抄錄完整物價表（依出版社版權）。

🌐 6 國總覽

① 神聖吉札帝國

政體：帝國（神權系）

主要城鎮：歐德普克（首府）/ 路特卡得

特色：神聖系寺院 / 暗黑系寺院並存，整套訓練所（劍 / 斧 / 棍 / 弓 / 投 / 盜賊 / 野營 / 物理魔法）

隱藏秘寶（5 件神器）：猛驍佳神（巨大單手刀）/ 聖封誤錄 / 太陽神弓（黃金弓）/ 巨人護手 / 歐利哈肯（東方魔幻金屬唐錘）

特產品：玻璃裝飾品（33,200G）/ 水銀登墨陶器（31,950G）

建議玩家階段：早中期，新手起點

② 敦津（中華風格，歷史深邃）

政體：地方城邦（中華風格）— 街市展現中國般風格，有許多遺跡與洞窟

主要城鎮（5 個）：江都 / 廣南 / 華安 / 天陽 / 仙醉鄉

特殊地點（13 個）：月下遺跡 / 水上遺跡群 / 道幻之町 / 鎮乳洞 / 寬之遺跡 / 院亡地下遺跡 / 拌之之遺跡 / 進口森林 / 地下遺跡群 / 寬寬之道跡 / 鬼之神殿 / 莨皇建築物道跡 / 普利桓殿

特色：江都的訓練所最完整（含元素系魔法）；華安偏港口貿易；中級武器及防具（神聖吉札冒險者來此強化裝備）

建議玩家階段：中期

③ 樸哈爾帝國（廣大土地 / 河川 / 乾燥區 / 石頭洞窟）

政體：帝國（西亞 / 阿拉伯風格）— 為達成任務常需走過相當長的距離

主要城鎮（7 個）：阿薩拉姆 / 烏魯姆 / 埃耳卡拉亞 / 樸夸利卡 / 嘉哈 / 索爾哈拉亞 + 1 個

地下城（17 個）：古迷宮 / 議廷遺跡 / 砥砥之墓 / 試練迷宮 / 大學殿遺跡 / 象牙塔 / 大宮殿 / 砥之之地 / 廣坑 / 魔家研究所 / 地下神殿 / 地下審判迷宮 / 水上神殿遺跡 / 蘿之蕃塔 / 山賀大洞窟 / 鐵練馬遺跡 / 不流澤布岡蹬

特色：道具店有特產（香料 / 絨毯 / 象牙 / 扇葉）；神聖 + 暗黑系並存

阿薩拉姆（首都）：訓練所 8 技能（劍 / 槍 / 弓 / 格鬥 / 迴避 / 盜賊 / 野營 / 元素系魔法）+ 神聖系寺院 + 暗黑系寺院 + 城（通行證：敦津 20,000G / 馬尼卡帕王國 25,000G）+ 魔法屋含守護娃娃 25,000G + 特產店（香料 24,200G / 絨毯 32,400G）

烏魯姆完整設施：武器店（劍 3,000G / 大劍 5,000G / 旭弓 8,700G / 牛角箭 9,620G 等）+ 魔法屋（寶石戒指 20,000G / 解毒藥 80G / 復生石 520G 等）+ 特產店（香料 24,200G）+ 暗黑系寺院 + 驛馬車（→阿薩拉姆 3 日 1,500G / →樸夸利卡 4 日 2,000G / →嘉哈 5 日 2,500G）

索爾哈拉亞：訓練所（劍 / 棍棒 / 槍 / 投擲 / 盜賊 / 元素系魔法）+ 特產店（象牙 31,900G）+ 驛馬車（→埃耳卡拉亞 2 日 1,000G）

嘉哈：暗黑系寺院 + 特產店（扇葉 23,300G）+ 驛馬車（→烏魯姆 5 日 2,500G）

建議玩家階段：中期

④ 日傑（劍家路線推薦國家，4 個臨海部分）

政體：地方諸國（日式風格）— 周圍臨海，國土分為四個部分，是個小國

主要城鎮（5 個）：紫波羅 / 鷹卑 / 慶谷 / 戴譴野 / 額蔚

地下城（12 個）：鬼地 / 巨咒迷宮 / 博物館遺跡 / 透途洞窟 / 地下基地 / 鎮下迷宮 / 大洞窟 / 鼠丹遺跡 / 黑闇城 / 沉迄水路 / 地下神殿 / 命還神殿

強裝介紹：餌太黃（最強之劍）— 劍家路線必訪

特色：道具店有特殊道具（松茸 / 妖刀 / 銅鎌）；訓練所偏元素系魔法 / 用劍高手聚集

建議玩家階段：中後期

⑤ 馬尼卡帕王國

政體：王國（中亞 / 印度風格，土地大部分被密林覆蓋）

主要城鎮（5 個）：集斯塔利貝拉 / 聖米哥 / 摩爾尼加 / 埃爾門德 / 里歐庫列歐

地下城（11 個）：質賈遺跡 / 地下基地 / 留麗瑪達跡 / 蝶怪遺跡 / 地下遺跡 / 證持神殿 / 悉吾遺跡 / 鋼上慰塔 / 狂魚倍依遺跡 / 神殿遺跡 / 鋼影遺跡 / 魔神壁之店

特色：強力武器與防具（玩家應走一趟充實裝備） / 港口連接外國 / 高階魔法物品 / 特產品蜂王乳 / 水晶

建議玩家階段：後期

⑥ 芙庫基歐（精靈魔法之國，最終區域）

政體：邊境地區（最後解鎖）— 世界上唯一有教授精靈魔法訓練所的國家

居民特色：身上長著翅膀（奇妙的國家）

主要城鎮（3 個）：帕爾曼吉 / 漢爾蓋油 / 緻野

地下城（8 個）：儀式之塔 / 鐵之風塔頭 / 巨摩塔 / 妖狀痕廢盧 / 不動之迷宮 / 鈴幻噴星跡 / 順上之原遺跡 + 1 個

⚠️ 航運警告：要注意四、八、十二月船不出航

設計建議：戰士系 / 僧侶系角色推薦造訪（裝備品有優秀魔法抵抗力 + 精靈系魔法訓練）

建議玩家階段：終局

🚂 驛馬車網路結構

驛馬車有三種計費模式：

  • 近城短程：2-3 天 / 1200-1500G — 同國城際
  • 跨國中程：4-7 天 / 2400-4000G — 鄰國移動
  • 跨海長程：8-20 天 / 4000-10000G — 跨大陸 + 配合港口

港口連接（需要海運的城）：

  • 埃耳卡拉亞 ↔ 皇歐庫列歐（樸哈爾 ↔ 馬尼卡帕）
  • 多特貝克 / 阿拉伯（敦津華安 ↔ 樸哈爾）
  • 埃耳卡拉亞 ↔ 里歐庫列歐（馬尼卡帕內 + 跨國）

🎯 訓練所提供的 9 大基礎技能

從攻略書記錄，訓練所典型提供：

  1. 劍（劍系武器）
  2. 斧 / 大斧（斧系）
  3. 槍（長槍）
  4. 棍棒 / 榴棒（棍系）
  5. 弓（射擊）
  6. 投擲（飛標 / 矢）
  7. 盜賊（解鎖 + 偷竊）
  8. 野營（戶外維生）
  9. 物理系魔法 / 元素系魔法（部分訓練所）

進階：神聖系魔法在「神聖系寺院」學；暗黑系魔法在「暗黑系寺院」學；精靈系魔法需要特定的精靈系術屋。 完整 5 系魔法請見 5 系魔法施展指南（同步出版於本波）。

📖 玩家路徑建議（基於本站資料庫整理）

  1. 新手起點：神聖吉札帝國 歐德普克 → 學基礎武器 + 物理系魔法
  2. 初期擴展：路特卡得 → 敦津江都 → 學元素系魔法
  3. 中期跨國：港口跨海到馬尼卡帕王國 → 學高階魔法
  4. 後期：日傑 → 龍諸野 → 學暗黑系魔法 / 必殺技
  5. 終局：芙庫基歐 → 攻略書 PART 5 boss 群（魔王阿迦魯瑪 Lv 50 / HP 999）
