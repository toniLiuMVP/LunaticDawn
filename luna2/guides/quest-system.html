📜 任務系統概要

俠客遊 II 是開放世界 RPG，玩家透過接任務賺取金錢 + 經驗值 + 物品。 任務從攻略書 PART 3「進入冒險的世界」和 PART 4「淬煉後的英雄」分類為 5 大類型 + 3 條主路線。 本攻略整合任務種類、訓練系統、Save/Load 技巧、危險區須知。

🎯 5 大任務類型

① 送貨任務

Easy

內容：將指定物品從 A 城送到 B 城。

  • 難度：低（多在 2-3 天內完成）
  • 報酬：500-3000 G
  • 風險：途中可能遇盜賊 / 怪物搶劫
  • 建議：新手期主要收入來源

② 送信任務

Easy

內容：將信件送給特定 NPC（多在不同國家之間）。

  • 難度：低-中（需跨國移動）
  • 報酬：800-2500 G
  • 風險：信件遺失（被搶 / 死亡）= 任務失敗
  • 建議：搭配跨國貿易最划算

③ 護衛任務

Mid

內容：保護指定 NPC 安全到達目的地。

  • 難度：中（NPC 戰力弱需保護）
  • 報酬：2000-8000 G
  • 風險：NPC 死亡 = 任務失敗
  • 建議：適合有同行隊友 + 裝備齊全的中期玩家

④ 救援任務

Hard

內容：救出被囚禁 / 失蹤的 NPC（從迷宮 / 危險區）。

  • 難度：高（需進迷宮 + 戰鬥）
  • 報酬：5000-25000 G
  • 風險：迷宮怪物強 + NPC 在多層深處
  • 建議：用「索敵術」「尋寶術」（物理系魔法）輔助

⑤ 暗殺任務

Evil Path

內容：暗殺特定 NPC（多在邪惡之路）。

  • 難度：高（目標通常戰力強 + 有護衛）
  • 報酬：10000-50000 G
  • 風險：失敗會被通緝 / 名聲下降
  • 建議：適合「殺人高手」「暗影支配者」「死神」等暗黑路線

🎓 訓練術法事半功倍

每國訓練所提供基礎技能訓練。攻略書 PART 3 ch1 列出訓練成本：

訓練等級等級 1-4等級 5-8等級 9-12等級 13-16
每級費用250 G~30 G/8 級~6 萬 G/級10000 G/級+
升 1 級時間即時1-3 天3-5 天7+ 天
建議階段新手期初期中後期後期 + boss 戰前
訓練 ROI 策略：
  • 新手期：5-8 級每級只 30 G，CP 值最高 — 主武器類別早早練到 8 級
  • 中期：暫停武器 / 開始魔法系訓練（魔法屋）
  • 後期：選擇練「弱項補強」或「強項極致」 — 後者 ROI 較高
  • 等級 13-16：搭配神器級武器倍率最高

訓練的 3 大術法系統（PART 3 ch1）：

  • 魔法系（劍 / 槍 / 棍 / 弓 / 投擲 / 拳）— 訓練所教練
  • 信仰系（神聖 / 暗黑 / 精靈魔法）— 寺院 / 精靈術屋
  • 神祕系（物理 / 元素魔法）— 魔法屋

💾 Save/Load 技巧（攻略書 PART 3 ch1 提到）

俠客遊 II 是 1996 年 DOS RPG，沒有自動存檔！ USER0 ~ USER9（10 個存檔槽）需要玩家手動 save。

① 主要存檔時機

  • 進入新地區 / 新國家前
  • 接到重要任務後（避免任務失敗重做）
  • 進迷宮前（boss 戰前必存）
  • 等級提升後 / 學新魔法後
  • 大筆交易 / 招募 NPC 後

② Save/Load 心法

  • 多 slot 並用：USER0 主存檔，USER1-USER3 階段備份
  • 不要單一 slot 反覆覆蓋：避免存到「壞掉的存檔」回不去
  • 進迷宮前 save 到 USER1，主存檔保留在 USER0
  • 使用本站屬性修改器需先 save 到 USER1+，避免修改器出錯影響主存檔

🗺️ 危險區冒險須知（攻略書 PART 3 ch2）

① 迷宮探險前準備

  • HP / MP / STM 全滿
  • 解狀態異常道具：解毒石 ×3 / 解沈靜符 ×3 / 防盲目鏡片 ×2
  • 恢復道具：療傷藥 ×5 / 高級療傷藥 ×3 / 回復箱 ×1
  • 學「照明術」（地下城用）+ 「索敵術」（顯示敵人位置）+ 「尋寶術」（顯示寶物位置）
  • 同行 NPC ≥ 2 個（戰士 + 治療類）

② 進入迷宮後

  • 每樓層 save 一次（防止 dead 全 lose）
  • 記錄已走過的地圖（俠客遊 II 沒有自動 mini-map）
  • 遇到強敵先逃跑 / 用「停時術」逃離
  • 注意陷阱（地刺 / 落穴 / 毒霧）

③ boss 房常見模式

  • boss 通常在迷宮最深層
  • 進房前確認 HP/MP/STM 全滿
  • boss 可能有護衛（先清護衛）
  • 戰鬥外可用戰鬥策略整合篇對應 boss 戰術

🛤️ 三條路線的任務分布

① 英雄之路（PART 4 ch1）

主要任務類型：救援 / 護衛 / 冒險探索 / boss 討伐

建議流程：

  1. 新手期：神聖吉札帝國 — 接送貨 / 送信
  2. 初期：路特卡得 — 護衛任務
  3. 中期：跨國救援 — 從迷宮救出 NPC
  4. 後期：討伐 boss 級怪物（賺英雄聲望）
  5. 終局：芙庫基歐 boss 群

NPC 建議：戰士 / 武術高手 / 神聖魔法高手

② 邪惡之路（PART 4 ch2）

主要任務類型：暗殺 / 偷竊 / 走貨小販 / 邪惡 NPC 招募

建議流程：

  1. 新手期：偷竊小東西練盜賊技
  2. 初期：加入邪惡集團 — 找一位「邪鬼吉」級邪惡 NPC 招募
  3. 中期：接「暗殺」任務（高報酬）
  4. 後期：暗黑系魔法 + 「殺人高手」NPC 隊伍
  5. 終局：黑暗系 boss + 邪神結局

NPC 建議：殺人高手 / 黑暗帝王 / 死神 / 破滅之徒

注意：邪惡之路後不能輕易回頭。多 slot save！

③ 金錢之路（PART 4 ch3）

主要任務類型：送貨 / 送信 / 跨國貿易 / 賞金獵人

建議流程：

  1. 新手期：神聖吉札短程送貨
  2. 初期：跨國貿易（樸哈爾香料 ↔ 日傑松茸）
  3. 中期：賞金獵人路線（CALLNAME #98）
  4. 後期：高 G 暗殺任務 + 自開店
  5. 終局：富甲一方，購買神器級武器

NPC 建議：商人 / 吟遊詩人 / 賞金獵人 / 賢者

💡 任務系統的隱藏細節

① 任務隨機性：俠客遊 II 任務多隨機產生（從 ENC 字典抽取對象 + 報酬範圍）。 不同存檔遇到的任務都不同 — 有「重 roll」的策略空間。

② 名聲系統：完成任務累積名聲 — 名聲影響後續可接任務的等級 + NPC 招募成功率。

③ 任務時限：部分任務有日期限制（送貨 5 天 / 送信 7 天）。錯過 = 失敗 + 名聲下降。

④ 過期任務：超過時限會在「OVERDUE.ENC」記錄（已從 binary 解出 1 個 ENC）。

🎯 玩家進階技巧

  1. 任務疊加：同方向接多個任務（A 城送貨 + 同城送信 + 護衛去同方向 NPC）— 一趟完成 3 個
  2. 偷雞時間 / 慢遊戲：時間任務的緊張感 — 玩家可選擇「慢遊戲」推遲時限換更多探索
  3. 跨國貿易：金錢之路重點 — 樸哈爾香料 / 絨毯 / 日傑松茸 / 妖刀 / 馬尼卡帕蜂王乳 / 水晶 各國價差
  4. 名聲管理：英雄路線高名聲 / 邪惡路線負名聲；商人路線中性 — 三條路線可並行但要管理好
