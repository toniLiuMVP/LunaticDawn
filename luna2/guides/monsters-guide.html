📜 怪物系統概要

俠客遊 II 共有 128 隻怪物（id 0-127）存於 P/MONSTER.DAT，每記錄 104 bytes 含 13 個欄位（HP / MP / STM / Lv / ATK / 經驗 / 金錢 / 掉寶 ×2 / 防砍刺敲 等）。 本攻略基於第八/十三/十四波 binary 完整逆向 + 攻略書 PART 5 ch3「怪物資料大全」cross-validate。

🐺 怪物分類（按等級階段）

① 新手期（Lv 3-7）id 0-30 區

人類組（盜賊 / 戰士 / 武士 / 騎士 等）+ 部分動物類

  • 人類組：強盜 / 忍者 / 殺手 / 戰士 / 魔術師 / 祈禱師 / 盜賊 / 獵人 / 騎士 / 武士 / 僧侶 / 尼姑 / 狂戰士 / 劍鬥士 / 勁裝戰士 / 吟遊詩人
  • HP 範圍：60-160
  • ATK 範圍：32-82
  • 建議裝備：寶劍刀類 / 雙手刀類，加 1-2 件護具

② 中期（Lv 10-25）動物 / 龍蛇類

動物類（大蝙蝠 / 大兔牛 / 巨蛇 / 貓兒）+ 龍蛇類（蜥蜴 / 人面蛇 / 巨蛙 / 蛇蝎女妖）

  • HP 範圍：100-300
  • 特殊狀態：部分有「中毒」「石化」攻擊（用解毒石 / 解石符）
  • 建議：學物理系魔法的「沉默術」「停時術」克制 spawn 多的動物群

③ 中後期（Lv 25-40）魔法系 + 巨人類

魔法系（魔魂兒 / 敏蝠魔 / 酸魔魂 / 半人馬 / 半蛇人 / 石膏蝠）+ 巨人類（蜘蛛 / 石巨蛇 / 大巨人 / 龐巨人）

  • HP 範圍：200-500
  • 魔法防禦：學「封魔術」減低魔法系 dmg
  • 建議：必殺技「真槍貫術」「直棍撲術」破甲

④ 後期（Lv 35-45）不死 / 妖精 / 昆蟲

不死生物（殭屍人 / 髑髏兵 / 屍狼 / 不死之主）+ 妖精（精靈 / 鬼魂 / 巨蜂 / 闇龍）+ 昆蟲（蜂霊 / 毒火 / 森林女神 / 林霊騎士）

  • HP 範圍：300-700
  • 不死類：神聖系魔法「除魔大法」一招擊潰
  • 妖精類：物理攻擊較弱，用元素系魔法（爆炸 / 隕石）

⑤ 終局（Lv 45-50）龍類 + 魔王群

龍類（雙頸龍 / 傳龍 / 火龍 / 死之龍 / 黑龍 / 邪火龍 / 赤龍）+ 魔王群（id 122-127）

  • HP 範圍：500-999
  • 特殊技：boss 級用必殺技 + 多重屬性攻擊
  • 建議：學齊 5 系魔法 + 必殺技變體形態 + 神器級武器（火炎神斷 / 神切劍）

🛡️ 弱點屬性對照

俠客遊 II 怪物有 7 種弱點類型（從 LUNA2.EXE 偏移 0x040DCC 提取確認）：

弱點對應怪物類別有效攻擊
野獸恐懼症動物類物理攻擊 + 元素系
魔獸恐懼症魔法系怪物 / 龍類物理攻擊 + 神聖系
魔法生物恐懼症魔魂兒 / 半人馬 / 妖精類物理攻擊 + 暗黑系
巨人恐懼症巨人類必殺技「真槍貫術」/「直棍撲術」
不死生物恐懼症殭屍 / 髑髏 / 屍狼 / 不死之主神聖系「除魔大法」
幽靈恐懼症鬼魂 / 闇龍神聖系魔法
沒有無弱點（多為 boss 級）需高等級 + 多種策略

🔥 怪物使用必殺技（HISSATSU.MST records 16-31）

怪物使用的 16 個特殊類必殺技：

  • 12 common（id 16-21, 26-31）：b1=60，威力 250-500，無玩家動畫
  • 4 low_cost（id 22-25）：b1=15 或 10，威力 100，自動觸發

怪物資料庫的「必殺技 O/X」欄位標 O 的怪物會用這些必殺技。多數低階怪物（人類組 / 動物類）標 X，**只有 boss 級 + 部分龍類 / 魔法系**才有必殺技。

對抗 boss 必殺技的策略：
  1. 識別 boss 必殺技的「目標」 — 全體 / 單體 / 直線
  2. 用「沉默術」（物理系）阻止施法型 boss
  3. 用「停時術」延遲 boss 動作
  4. 分散站位避免全體必殺技

💰 經驗值與掉寶

MONSTER.DAT 13 欄位含 EXP / 金錢 / 掉寶 ×2，常見模式：

  • 新手期：EXP 10-30，金錢 50-200
  • 中期：EXP 50-100，金錢 500-2000
  • 後期：EXP 100-150，金錢 2000-5000
  • boss 級：EXP 150-170，金錢 / 掉寶有特殊高階物品

具體每隻怪物的 EXP / 金錢 / 掉寶請查 怪物資料庫（128 records 完整數據）。

🎯 玩家路線建議

  1. Lv 1-10：神聖吉札帝國練人類組怪物，學基礎武器 + 物理系魔法
  2. Lv 10-20：跨國去敦津 / 樸哈爾，打動物 / 龍蛇類，學元素系魔法
  3. Lv 20-30：日傑練魔法系怪物 + 巨人類，必殺技變體形態解鎖
  4. Lv 30-40：馬尼卡帕王國學神聖 / 暗黑系，打不死類 + 妖精
  5. Lv 40-50：芙庫基歐學精靈系召喚獸，準備挑戰魔王群
  6. Lv 50：挑戰提斯卡托（最終 boss，HP 999）

🔬 技術細節（給逆向有興趣的人）

MONSTER.DAT 結構（每 record 104 bytes）：

  • 0x14: HP（u16 LE）
  • 0x16: MP（u16 LE）
  • 0x18: STM 體力（u16 LE，第十三波 PDF cross-validate 確認）
  • 0x1A: 等級
  • 0x24: 經驗值
  • 0x28: 金錢
  • 0x3E: ATK 攻擊力
  • 0x40-0x44: 防砍 / 防刺 / 防敲

parser + cross-validate 程式：tools/extract/cross_validate_monsters.py
