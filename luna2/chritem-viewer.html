CHRITEM.SAV 物品欄查看器

← 屬性修改器 · 回主站
俠客遊 II 玩家物品欄存於 USER0/CHRITEM.SAV（122880 bytes）。 本工具支援 查看 + 編輯（v0.2 2026加入 edit 模式），可解析 160 個角色（玩家 + NPC 同行）的物品欄。

⚠️ 修改前務必備份：本工具的 byte 0-7（item_id + value2）已 binary 逆向確認， 但 byte 8-47（modifier / 強化值）未實機驗證。修改後若載入遊戲出錯可能損壞角色。 建議流程：
1️⃣ 將 USER0/CHRITEM.SAV 先複製為 USER1/CHRITEM.SAV（DOSBox 內可選 USER1 = 存檔 2）
2️⃣ 在 USER1 測試修改 + 載入遊戲驗證
3️⃣ 確認 OK 後再覆蓋 USER0
（吸收2026 round 4 RedTime「寫工具與工具不同步」教訓）

結構（2026-05-02 2026逆向）：160 角色 × 16 物品 slots × 48 bytes/slot。 每 slot：byte 0-3 = item_id（對應 ITEMNAME.ENC 的 1-456 物品）；byte 4-7 = value2（推測為價格）； byte 8-47 = 強化值 + 屬性 modifier + 額外資料。

使用方式：拖放或選擇 USER0/CHRITEM.SAV 檔案 → 左側選角色 → 右側看物品欄。 所有解析在瀏覽器本地完成，不上傳任何資料。
