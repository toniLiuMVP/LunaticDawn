LUNA2 DATABASE · 技術研究

本站為俠客遊 II DOS 版二進制檔案格式的獨立逆向研究。

公開頁面僅展示「技術知識」（檔案格式、加密演算法、struct 結構）與「逆向出的數值」（價格、HP、攻擊力等可量化數據）。原版受著作權保護的創意內容（圖像、劇情文字、完整名稱資料）不在公開頁面展示。

▸ ENC 加密：4-byte XOR 演算法（key 為運作時硬編碼常數）
▸ MST/DAT 結構：fixed-record 切片，u16/u8 little-endian struct
▸ 純前端、純 DOM 渲染、無 innerHTML、無框架
▸ 公開版本說明
依 PUBLISHING.md 規範，items / monsters / magics / hissatsu / dungeons 五類資料的「名稱欄位」目前以「(本機保留)」顯示。所有數值欄位（HP、價格、攻擊力等）完整保留。

▸ PLT 調色盤格式

.PLT 檔案編碼：每色 4 bytes（u16 LE GRB444 + 2 bytes alpha），256 色 palette，PC-98 風格 12-bit RGB 排列（高 4 bits=綠 / 中 4 bits=紅 / 低 4 bits=藍）。

其他場景 palette 與所有 DDT 圖像衍生物本機保留，不在公開頁面展示。
