LUNA2 DATABASE v0.1

從 1996 年 Artdink《俠客遊 II》DOS 版遊戲本體逆向解析出的完整資料庫。

所有名稱與數值來自遊戲檔案 俠客遊2/P/*.MST 與 俠客遊2/O/*.ENC，非從攻略整理重製——是真正的「事實在二進制裡」。

▸ ENC 加密：4-byte XOR key 9A BC DE 00 循環，全 20 個 ENC 檔已解密
▸ MST 結構：依 record size 切片，u16/u8 little-endian struct 解析
▸ 純前端、純 DOM 渲染、無 innerHTML、無框架

▸ PALETTE GALLERY

.PLT 調色盤已逆向：每色 4 bytes，u16 LE GRB444 + 2 bytes alpha（PC-98 風格 12-bit 顏色）。 256 色 × 16×16 grid 顯示如下。

⏳ 待辦：用這些 palette 解碼 .DDT/.DD9/.PAC/.E32 圖像格式（含 12-byte filename header + 編碼像素資料）→ 怪物圖、物品圖示、角色立繪 PNG。

▸ SPRITE GALLERY (BETA)

.DDT 圖像格式部分逆向：mode 2 (4-bit EGA 4-plane separate) 已成功解碼。 header: filename(12B) + reserved(12B) + mode(u16) + width(u16) + height(u16) + reserved(u16)。 pixel layout: data = plane0全部 + plane1全部 + plane2全部 + plane3全部，每像素顏色 = bit0 | bit1≪1 | bit2≪2 | bit3≪3。

▸ mode 4/6/16 (戰鬥特效 + 道具)

確認 mode 4/6/16 也是 EGA 4-plane separate（同 mode 2 layout）。mode 數字不影響 pixel 排列，只影響 palette 對應。

⏳ 70+ DDT 已批次解碼。mode 12/13/14/15/18/24/25/35/36/39 仍待逆向（可能是不同 plane count 或壓縮格式）。

資料庫由 工具/extract/parse_mst.py 從遊戲二進制檔案產生。
