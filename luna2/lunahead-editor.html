LUNAHEAD.SAV 玩家表頭編輯器

屬性修改器 · 物品欄 · 商店 · SAV 總覽 · 回主站
俠客遊 II 玩家存檔表頭存於 USER0/LUNAHEAD.SAV（256 bytes）。 本工具支援編輯玩家姓名（Big5）+ 數值欄位 + 下載修改後 SAV。

結構（2026-05-02 2026 binary 逆向）：byte 0-17 = 玩家資料（含 Big5 姓名）+ byte 18+ 全 0xFF padding。
byte 5-10：Big5 玩家姓名（3 個中文字，6 bytes）— USER0 預設「未具名」。 byte 14-15 / 16-17：兩個 u16 LE = 48（推測為 age / 起始屬性）。

所有解析在瀏覽器本地完成，不上傳任何資料。

⚠️ 修改前務必備份：本工具的 byte 5-10（Big5 玩家姓名）+ byte 14-17（u16 LE） 已 binary 逆向確認，但 byte 1-4（推測為 flag / 日期欄）未實機驗證。修改後若載入遊戲出錯可能損壞存檔。 建議流程：
1️⃣ 將 USER0/LUNAHEAD.SAV 先複製為 USER1/LUNAHEAD.SAV
2️⃣ 在 USER1 測試修改 + 載入遊戲驗證
3️⃣ 確認 OK 後再覆蓋 USER0

▶ 拖放 LUNAHEAD.SAV 到這裡
