[ ABOUT ]

俠客遊 II 是 1996 年的 DOS 遊戲。原本要在 macOS / Windows 11 / Linux 上跑起來，得自己裝 DOSBox、改 config.sys、 凹出 LH + UMB 把 DOS 載到高位記憶體、解決 640 KB 常規記憶體 不夠的問題，還要手動設音效卡 IRQ/DMA——每一關都能勸退 90% 的人。

這個啟動器把那些 1996 年的地獄一次處理好——下載 3 個檔案、 雙擊，就可以跑。跨 macOS / Windows / Linux。

※ 本站不提供遊戲本體下載（版權因素）。請自備 LUNA2.EXE 及原版資料檔。

[ STEP 1 · 先裝 DOSBOX-X ]

DOSBox-X 是 DOSBox 的一個分支，對舊 DOS 遊戲的相容性較好，特別是中文顯示、音效 TSR、記憶體管理都比原版強。

macOS

brew install --cask dosbox-x-app

沒有 Homebrew 可以到 dosbox-x.com 下載 .dmg 手動安裝。

Windows

winget install DOSBox-X.DOSBox-X

或到 dosbox-x.com 下載 .exe 安裝檔。

Linux

# Debian / Ubuntu
sudo apt install dosbox-x

# Arch
sudo pacman -S dosbox-x

# 或從 GitHub Release 下載 AppImage

※ 需要 DOSBox-X 0.84 或以上。舊的 DOSBox（非 X 版）也能跑，但需要改設定。

[ STEP 2 · 下載啟動器 ]

下載 dosbox-x.conf（必備）+ 你 OS 對應的啟動器 （Mac 下載 .command，Windows 下載 .bat，Linux 下載 .sh）。

▼ dosbox-x.conf ▼ 啟動俠客遊II.command (macOS) ▼ 啟動俠客遊II.bat (Windows) ▼ 啟動俠客遊II.sh (Linux)

※ 瀏覽器可能會把 .command / .bat 當成危險檔案。放心，原始碼都在 GitHub Repo 可以審查，每一行都看得見。

[ STEP 3 · 放進遊戲資料夾並執行 ]
3.1 把下載的 3 個檔案（dosbox-x.conf、 啟動俠客遊II.command 或 啟動俠客遊II.bat） 全部放在和 LUNA2.EXE 同一個資料夾。
3.2 · macOS 使用者 第一次執行時右鍵點 啟動俠客遊II.command → 選「打開」→ 彈出警告 → 選「打開」。

或者終端機跑一次： 
chmod +x 啟動俠客遊II.command
之後雙擊即可。
3.2 · Windows 使用者 直接雙擊 啟動俠客遊II.bat。 Windows Defender 可能會警告「已防止無法辨識的應用程式啟動」， 點「其他資訊」→「仍要執行」。
3.2 · Linux 使用者 終端機先給執行權限： 
chmod +x 啟動俠客遊II.sh
./啟動俠客遊II.sh
或者在檔案管理器（Nautilus / Dolphin / Thunar 等）右鍵 → 「屬性」→ 勾選「允許作為程式執行」，之後就可以雙擊 →「在終端機中執行」。
3.3 DOSBox-X 視窗跳出 → 看到 綠色開場白 → 進入俠客遊 II → 選擇 USER 存檔槽 → 開始遊戲。

[ 常見問題 ]

Q1. 為什麼要用 DOSBox-X 而不是 DOSBox？

DOSBox-X 相容性比原版強很多——支援中文 Big5 顯示、完整的 EMS/UMB/HIMEM 模擬、 SVGA 模式、更精細的 CPU 週期控制、以及 dos = high,umb 這種模擬 MS-DOS 6.22 config.sys 的能力。俠客遊 II 這種 1996 年的繁中遊戲，DOSBox-X 是現在最穩定的選擇。

Q2. 遊戲跑太快 / 太慢？

按 Ctrl + F11 減速，Ctrl + F12 加速。 預設是 20000 cycles（約 Pentium 100 等級），對俠客遊 II 來說剛好。

Q3. 沒有音效？

俠客遊 II 的音效需要先執行 M\SB16.COM 載入 DIGPAK TSR 驅動。 本啟動器為了相容性先不載入音效驅動——大部分人其實玩的是無聲版。 需要音效的玩家可以在 DOSBox-X 裡手動執行： 

LH MIDPAK.COM
LH M\SB16.COM
LUNA2.EXE

（LH = LoadHigh，把 TSR 載到高位記憶體避免吃掉 640 KB 常規記憶體）

Q4. 存檔在哪裡？

存檔是 USER0 到 USER9 這 10 個資料夾（對應遊戲裡的 10 個存檔槽）， 每個資料夾裡有 13 個 .SAV 檔案。

想修改存檔？試試看本站的 ▸ 瀏覽器版修改器——直接拖 LUNACHAR.SAV 進去就能改。

Q5. 找不到 LUNA2.EXE？

確認遊戲資料夾裡面有 LUNA2.EXE（主程式，大約 610 KB）。 如果是剛從 CD 安裝，執行過 INSTALL.EXE 應該就會自動放好所有檔案。

Q6. 啟動器無法執行（Mac 說「未識別的開發者」）？

這是 macOS Gatekeeper 的保護。解法： 

chmod +x 啟動俠客遊II.command
xattr -d com.apple.quarantine 啟動俠客遊II.command

或者到「系統設定 > 隱私權與安全性」最下面允許執行。

[ 技術細節 ]

啟動器的運作原理——對技術有興趣的人可以看。

dosbox-x.conf 做了什麼？

這個 conf 檔設定：

  • memsize = 16：給 DOS 虛擬 16 MB 記憶體
  • machine = svga_s3：模擬 S3 Trio 顯卡（支援 SVGA 640×480 真彩色）
  • cputype = pentium, cycles = 20000：模擬 Pentium 100 等級
  • sbtype = sb16：Sound Blaster 16（IRQ 7, DMA 1/5）
  • xms = ems = umb = true：開啟所有記憶體擴充功能
  • ver = 6.22：模擬 MS-DOS 6.22 報版本
  • customcodepage = 950：Big5 code page（避免 "No translation support" 警告）

啟動腳本做了什麼？

.command（Mac）/ .bat（Win）/ .sh（Linux）三個腳本做的事情一樣：

  1. 切換到腳本自己所在的資料夾（所以放哪都能跑）
  2. 檢查 DOSBox-X 是否存在（常見路徑 + PATH）
  3. 檢查 LUNA2.EXE 和 dosbox-x.conf 是否都在
  4. 執行 dosbox-x -conf dosbox-x.conf -c "mount c ." -c "c:" -c "LUNA2.EXE"

關鍵是 -c 參數——DOSBox-X 會把這些指令附加到 [autoexec] 區段後面執行。這樣 dosbox-x.conf 完全不用硬編路徑，任何人下載都能直接用。

為什麼不內建音效自動載入？

1996 年的 DIGPAK TSR 驅動（M\SB16.COM）對 DOSBox-X 的 SB16 模擬器並不是 100% 相容——有些 DOSBox-X 版本會在載入 DIGPAK 時當機。 為了讓「雙擊就能跑」的保底體驗穩定，預設不載入音效驅動。 音效測試版（dosbox-x-sound.conf）之後會放出來當進階選項。
