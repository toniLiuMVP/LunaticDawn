Lunatic Dawn IV (2001) · 次元之門與連線系統 · 台灣第三波代理

俠客遊 IV（Lunatic Dawn IV）是 2001 年由台灣第三波代理發行的系列作品，加入了次元之門與 IP 連線功能。本站收錄 toni 的連線設定方法、鞠躬練功法，以及 Morrowind 作的 LunaticConsortia 輔助工具三個歷史版本。

⚠️ 第三波官方伺服器於 2007 年前後關閉，現代要連線需透過 IP 直連方式。詳見下方「連線方法」攻略。

[ 光碟結構分析（2026 逆向）]

2026 年 toni 將實體光碟拷貝到電腦後，本站對 LD4 完整光碟結構做了二進位分析：

主程式

  • LD4.exe — 2,561,024 bytes（約 2.5 MB），編譯日期 2001-05-25，PE32 格式（Windows 32-bit i386 GUI）
  • 跟 LD3（2000）僅相差 6 個月發行；採同樣 MFC 框架但加入連線功能
  • 遊戲資料總大小約 334 MB（比 LD3 多 60 MB，多在 Character 跟 Effect）

★ Web/ 模板系統（LD4 獨有）

LD4 內建 Web/Models/ 4 種主題的 HTML 模板，玩家世界檔上傳到伺服器後會生成對應 HTML 頁面：

  • standard 標準主題：title 寫「LD IV WORLD INFORMATION」
  • colors / games / users：不同視覺主題
  • 每個主題含 5 個頁面：index / world / top / menu / player

從 Web/Models/standard/world.html 可看到玩家世界資訊頁的欄位設計：

  • HOME WORLD IP ADDRESS（核心 — 別人連你世界的進入點）
  • 世界名稱 / 公開狀態 / 開放狀況 / 容許人數
  • 訪客履歷：近 4 位訪客名單（World 名稱 + 玩家名稱 + 訪問時間 + 退出時間）

⚠️ 第三波官方伺服器於 2007 年前後關閉。但 IP 直連功能仍可用 — 詳見上方▸ 連線方法。

遊戲資料分類（Data/）

LD4 沿用 LD3 的 9 大類資料結構（系列共通設計）：

  • Map 107 MB — 地圖資料（最大）
  • Monster 82 MB — 怪物 sprite（增加大量新怪物）
  • Character 43 MB — 角色 sprite（比 LD3 多 19 MB）
  • Effect 8.1 MB — 戰鬥特效（比 LD3 多 3.7 MB）
  • Weapon / Shield 約 4 MB — 武器 / 盾牌資料
  • Campaign / Range / MSG — 劇本 / 範圍 / 訊息字典
  • usr/（LD4 新增）— 使用者存檔資料夾

⚠️ 怪物 / 角色 sprite 是創作性表現（屬版權保護內容），本站不公開。
