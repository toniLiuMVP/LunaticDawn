Lunatic Dawn IV (2001) · 次元之門與連線系統

俠客遊 IV（Lunatic Dawn IV）是 2001 年由台灣第三波代理發行的系列作品，加入了次元之門與 IP 連線功能。本站收錄 toni 的連線設定方法、鞠躬練功法，以及 Morrowind 作的 LunaticConsortia 輔助工具三個歷史版本。

俠客遊 IV頁面

▸ 俠客遊 IV 連線方法 第三波伺服器關閉後的 IP 直連方法：次元之門設定 → 界面選 IP → LAN 自動檢索 → 光球右鍵 → 輸入對方 IP。 作者：toni ▸ 俠客遊 IV 鞠躬練功法 兩個 NPC 老闆（酒館男 / 酒館女）+ 兩個劇本 + 兩個任務的交叉設定法。完成任務同時訓練隊友。 作者：toni
