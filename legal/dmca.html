[ 1. 本站的著作權合規哲學 ]

1.1 不公開創作性表現

本站不公開下列著作權保護內容:

  • 原版完整遊戲 EXE / ROM / 完整資料檔(玩家自備)
  • 遊戲內美術 / 角色立繪 / 場景圖
  • 遊戲內配樂(原 BGM 為玩家自備光碟 / Steam 安裝後的資料,本站不打包)
  • 遊戲內劇情對白原文
  • 攻略書 narrative 段落(本站只引用結構性事實 + 短引用 < 300 字 fair use 範圍)
  • 第三方 PDF / docx / 截圖檔

⚠ 注意例外:本站對 1996-2002 絕版作品的「相容性修補版 EXE」(字體修復 / Win10/11 相容性 / 跨平台 portable)有少數散布,詳見 §8 fan preservation patched EXE 特別說明。

1.2 公開事實層

本站公開的是依各國著作權法不受保護的事實層內容:

  • 🇹🇼 台灣著作權法 §10-2:單純為傳達事實之語文著作 / 事實 / 數據不受保護
  • 🇯🇵 日本著作権法 §2 第 1 項 第 1 號:創作性要件(事實 / 單純整理 ≠ 創作)
  • 🇯🇵 日本著作権法 §12-2:資料庫只保護創意性編排
  • 🇺🇸 17 USC §102(b):事實 / 發現 / 原則 / 過程不受著作權保護
  • 🇺🇸 Feist v. Rural Telephone (1991):事實 + 編排無創意性 = 非著作物
  • 🇺🇸 Sega v. Accolade (1992):逆向工程 disassembly = fair use

1.3 攻略文章 attribution

本站收錄的 2000-2010 年社群攻略(青衫主攻略 / 小蜜蜂 / 小傑 / Ertai / 聶荊璇 / germun / kyleul 等)均註明原作者。若你是這些攻略的原作者,認為 attribution 不正確 / 想撤除,請依下方流程寄信。

[ 2. 著作權異議流程 ]

2.1 必要資訊

寄信至 luntic.dawn@gmail.com 時請包含:

  1. 具體頁面 URL(本站哪一頁 / 哪一段)
  2. 你是著作權持有人或代理人(如:「我是青衫,某某 攻略原作者」/「我是 ARTDINK 代理人,代表 ARTDINK 主張權利」/ 「我是 某某 修改器原作者 kkt_zzz」)
  3. 你希望本站做什麼:
    • (a) 撤除該內容
    • (b) 變更 attribution(改名 / 加 link / 改說明)
    • (c) 要求加 disclaimer / 補充說明
    • (d) 其他
  4. 你的聯絡方式(toni 回覆用,本站不公開)

2.2 toni 的回應時程

  • 📨 收信後 7 天內 人工回覆(toni 為單人營運,非 24/7 客服)
  • 🛠 確認異議成立後 3 天內 修補 / 撤除內容
  • 📝 修補完成後寄信告知你,並提供 GitHub commit URL 證明

2.3 toni 不會爭辯

本站尊重原作者意願:

  • 你說撤就撤,不會問「你能不能改個說法」(雖然 fair use 法律上可能成立,但尊重你的意願優先)
  • 不會要求你提供詳細法律論證(信任你的身份聲明)
  • 不會引用 fair use 條款回駁(除非異議涉及惡意冒名 — 詳見 2.4)

2.4 例外:惡意冒名情況

若收到的異議信件明顯為冒名(例如有人冒充 ARTDINK 法務代表),toni 會:

  1. 要求進一步證明身份(公司信頭 / 律師資格 / 等)
  2. 必要時延後處理直到身份確認
  3. 不會單因匿名 / 不完整的異議信件就撤除合法內容

[ 4. 反通知 / Counter-Notice ]

本站不採用 DMCA 反通知機制 — 由於本站收到的異議信件主要來自原作者本人(非自動爬蟲產生的 false-positive 異議),不需要反通知保護機制。

若你認為某段內容被誤撤(例如有人冒名提異議導致你的合法收錄被撤),請直接寄信 toni 說明,toni 會審查重新上架。
