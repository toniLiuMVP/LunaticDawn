[ TL;DR ]

本站不收集你的任何個人資料。

  • ❌ 無 cookies
  • ❌ 無 Google Analytics / 任何 web analytics
  • ❌ 無第三方追蹤(no tracking pixels, no fingerprinting)
  • ❌ 無使用者帳號 / 註冊 / 登入
  • ❌ 無評論系統 / 不收集表單資料
  • ❌ 無廣告 / 無聯盟行銷 tracking

本站為 toni 維護的非營利數位文化保存專案,純靜態 HTML+CSS+JS 託管於 GitHub Pages。

[ 1. 本站運作模式 ]

「吟遊詩人的傳說 · 俠客遊」(https://toniliumvp.github.io/LunaticDawn/)是 100% 靜態網站,所有頁面為預先產生的 HTML 檔案,**沒有任何伺服器端程式碼**(無 PHP / Node / Python 後端)。

瀏覽本站時,你的瀏覽器只跟 GitHub Pages CDN(GitHub 提供的靜態檔案 host)通訊,**本站沒有自己的伺服器**。

[ 2. 你訪問本站時會發生什麼 ]

GitHub Pages 端(GitHub 平台,非本站可控):

  • GitHub 為了反 DDoS / 服務維護,可能會記錄你的 IP 位址跟訪問時間到 GitHub 伺服器 log(這是 GitHub 全平台政策,跟本站無關)
  • 詳見 GitHub 平台隱私政策

你的瀏覽器端:

  • 下載本站 HTML / CSS / JS / 字型 / 圖片 — 所有資源都託管在 GitHub Pages,**無第三方 CDN**(我們把 Google Fonts 本地化了,正是為了不洩漏訪客 IP 給 Google)
  • 存檔修改器頁面:你上傳的 .SAV 檔**完全在你的瀏覽器內處理,不會上傳到任何伺服器**(本站根本沒有伺服器)

[ 3. 你寫信給 toni 時 ]

本站數個頁面提供 mailto:luntic.dawn@gmail.com 連結讓你寄信給 toni。這些信件透過你的郵件客戶端 + Gmail 傳遞,**不經過本站**。

你寄出的信件:

  • toni 親自收讀,用於回覆你的問題
  • 不會分享給第三方
  • 不會用於行銷
  • 不會在公開頁面揭露你的 email / 真名(本站致謝段只用網路 handle 不公開真名 — 詳見 主站致謝段「跨海玩家回報」部分的處理方式)

若你希望 toni 不要保留或刪除你寄過的信件,請寄信告知 toni 會配合。

[ 4. 外部連結 ]

本站含多個外部連結:

  • Steam 商店(導流給 ARTDINK 正版購買):點擊連結會跳到 Steam,本站不參與分潤
  • Google Drive(toni 收藏的歷史檔案下載):點擊會跳到 Google Drive
  • GitHub(本站原始碼 + Issues):點擊會跳到 GitHub
  • 巴哈姆特 / PTT / 第三方論壇(歷史紀錄連結)

點擊外部連結後,你的瀏覽器跟對方網站直接通訊,**本站不會記錄你點了什麼**。你跳到的網站有他們自己的隱私政策,請自行閱讀。

本站所有外部連結均加 rel="noopener" 防止跳轉頁面取得 window.opener 引用(資安最佳實踐)。

[ 5. 兒童 ]

本站內容(俠客遊系列 1996-2002 老遊戲數位保存)沒有特別針對 13 歲以下兒童設計,但內容亦無不適合兒童的元素(無暴力 / 色情 / 賭博)。由於本站不收集任何個人資料,故 COPPA(美國兒童線上隱私保護法)合規性自動成立。

[ 6. 政策變更 ]

若本政策有重大變更(例如未來加入 analytics 或某種 tracking — 雖然 toni 無此計劃),會在主站「最新更新」區塊公告。本頁底部會顯示「最後更新日期」。
