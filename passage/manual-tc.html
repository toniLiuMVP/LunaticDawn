[ 第 1 章 · 遊戲安裝準備 ]

1-1. はじめに(序言)

ごあいさつ(問候):「本次承蒙惠購『ルナティックドーン~前途への道標~』,深表感謝。在遊玩本遊戲前,敬請務必詳閱本說明書。」

印刷方法(列印方法):從每頁說明選單的列印圖示可印出該節。

著作権について(著作權聲明):本說明書與遊戲內容之著作權屬 ARTDINK 公司所有。

動作環境(運作環境):Windows XP / Vista / 7 / 8 / 10 / 11 (Steam 復刻版相容性)。

マウスの操作方法(滑鼠操作):左鍵點選 / 右鍵選單 / 滾輪檢視 / 拖曳。

1-2. インストール(安裝)

インストールを行う(進行安裝):Steam 版自動安裝,1999 光碟版執行光碟內 SETUP.EXE。

インストールの手順(安裝步驟):

  1. 選擇安裝路徑(建議使用預設 C:\Artdink\Ld95)
  2. 選擇音效設定(BGM 開啟與否、音效介面卡)
  3. 等待檔案複製完成
  4. 建立桌面捷徑(可選)
  5. 完成 — 從開始選單啟動遊戲

⚠ Win10/11 玩家注意:原版 Ld95.exe 在現代 Windows 會有字體重疊問題。請套用本站 2026 字體修復補丁,拖檔即玩。

1-3. アンインストール(解除安裝)

Steam 版透過 Steam 用戶端右鍵 → 管理 → 解除安裝。1999 光碟版透過控制台「程式集和功能」移除。

玩家存檔(Usr/ 資料夾)不會被自動刪除,請依需求手動處理。

[ 第 2 章 · 開始遊戲 ]

2-1. ゲームの起動(遊戲啟動)

起動方法(啟動方法):從開始選單 → Artdink → ルナティックドーン 前途への道標 → 點選遊戲圖示。Steam 版透過 Steam 啟動,或執行安裝目錄的 Ld95.exe。

2-2. アプリケーションウィンドウ(應用程式視窗)

画面の説明(畫面說明):遊戲視窗分為視覺區、資訊區、地圖區、狀態區、工具列、狀態列、工具面板。

各メニューの説明(各選單說明):ファイル(檔案)/ パーティー(隊伍)/ ダンジョン(地下城)/ 戦闘(戰鬥)/ 設定 / ウィンドウ(視窗)/ ヘルプ(說明)。

2-3. ニューワールド(新世界 — 創角)

世界の創造(世界創造):選擇遊戲世界的勢力平衡、地理大小、初始種族分布等參數。

主人公(マイキャラクター)の設定(主角設定):

  • 姓名輸入
  • 性別選擇(男 / 女)
  • 種族選擇(人類 / 精靈 / 矮人 等)
  • 職業選擇(戰士 / 法師 / 盜賊 / 牧師 等)
  • 屬性點數分配(力量 / 體質 / 敏捷 / 智力 / 信仰 等)
2-4. セーブ・ロード(存檔・讀檔)

セーブ(存檔):選擇存檔槽(共 10 格)。建議在城鎮中存檔,避免戰鬥中存檔造成資料異常。

ロード(讀檔):從存檔槽選擇要讀取的遊戲進度。

存檔檔案位於遊戲安裝目錄的 Usr/ 資料夾。

[ 第 3 章 · 遊戲指引 ]

3-1. 世界の創造(世界設計)

勢力バランス(勢力平衡):遊戲世界由秩序、混沌、善、惡四大勢力組成。創角時可以設定各勢力強弱。

世界マップ(世界地圖):隨機生成的世界,包含多個城鎮、地下城、野外地形。

3-2. 町の役割(城鎮的功能)

宿屋(旅館):休息恢復 HP/MP,過夜推進時間。

武器屋・道具屋(武器店・道具店):販售武器、防具、消耗品。可賣出多餘物品。

寺院(寺院):治療異常狀態(中毒、麻痺等)、復活死亡夥伴、學習神聖魔法。

訓練所(訓練所):用金錢提升技能等級(劍術、弓術、魔法等)。

3-3. ダンジョンと戦闘(地下城與戰鬥)

ダンジョン(地下城):3D 第一人稱探索。可能有寶物、機關、怪物。

戦闘(戰鬥):回合制戰鬥。下達指令:攻擊 / 魔法 / 道具 / 防禦 / 逃跑。

3-4. そして、次の世代へ(然後,給下一個世代)

結婚と子供(結婚與孩子):與 NPC 結婚,生育子女。子女繼承部分父母屬性。

寿命と世代交代(壽命與世代交替):角色會老去死亡,玩家可選擇繼承給子女繼續遊玩。這是俠客遊系列獨特的「世代交替」機制 — 玩家經歷主角生老病死,然後接續其子嗣的人生。

[ 第 5 章 · 角色複製工具 ]

1997 原版沒有,2007 Steam 復刻版加入。獨立執行檔(キャラクターコピーユーティリティ.exe)允許將某存檔的角色複製到另一個存檔,方便玩家「再生」喜愛的角色,或在多週目遊玩時保留特定角色。

用法:

  1. 開啟工具,從左側選擇來源存檔。
  2. 選擇要複製的角色。
  3. 從右側選擇目標存檔。
  4. 點選「コピー」(複製)按鈕。
  5. 確認 → 完成。

⚠ 複製前建議備份兩個存檔(Usr/ 資料夾的對應 .sav)

站內替代:玩家 Cody 提供的「角色複製人物存檔包」可在 檔案庫下載,適合不會操作工具的玩家直接使用現成角色。

[ 第 6 章 · 使用者支援(原版內容已過時) ]

原版「正常に動作しない場合」對應的 2026 解法

原版說明書建議聯絡 ARTDINK 客服,但 2007 Steam 復刻版後,ARTDINK 已不再支援單獨技術問題。常見問題:

  • 画面がまったく表示されない(畫面完全不顯示):確認顯示卡驅動為最新版,DirectX 已安裝。
  • インストールができない(無法安裝):Steam 版自動處理。1999 光碟版需要光碟機 + 1.x GB 硬碟空間。
  • 突然動作しなくなった(突然無法運作):檢查 Usr/ 存檔是否毀損,從備份恢復。
  • 音楽、効果音が正常に再生されない(BGM、音效異常):Steam 版用內建 BGM(無光碟需求),1999 光碟版需光碟在光碟機中。
  • 字體重疊 / 字太小(Win10/11 限定):套用本站 2026 字體修復補丁。
ARTDINK サポート対応(原版客服資訊 — 歷史紀錄)

原 1997 年客服電話與信箱在 2007 Steam 版時已過期。如有 Steam 版技術問題,請透過 Steam 用戶端的「客戶服務」回報。
