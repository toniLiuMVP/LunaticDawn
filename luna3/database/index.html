俠客遊 III 隨機世界生成器名字字典（1,183 個 unique 中文字串）

來源：LD3 res/ 各 mt_*.BIN 跟 MonsterPara.BIN 內含的明文字串（無加密，magic \\0LD3）— 這些是 LD3 隨機世界生成器用的 name pool，每次新遊戲時系統從這些 pool 隨機選 world / town / dungeon / monster 名字。

合規：純名字字典，事實層內容（玩家於遊戲中會看到這些字串），依日本著作權法第 2 條僅創作性表現受保護，事實名稱不在保護範圍。

完整 JSON 在 ld3_world_pool.json。
