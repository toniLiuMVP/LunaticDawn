[ 光碟結構分析（2026 逆向）]

2026 年 toni 將正版遊戲實體光碟拷貝到電腦後，本站對 LD3 完整光碟結構做了二進位分析：

主程式

  • LD3.exe — 2,494,898 bytes（約 2.4 MB），編譯日期 2000-11-17，PE32 格式（Windows 32-bit i386 GUI）
  • 跟第三之書（2002）一樣是 MFC 程式，但 LD3 比第三之書早 2 年發行

光碟結構

  • 安裝光碟（LD3_INSTALL）：DirectX 安裝程式 + DirectXMedia + 主程式（ld3 資料夾）
  • 遊戲光碟（LD3_GAME_DISC）：Movie 動畫 + initial
  • 遊戲資料總大小約 275 MB

遊戲資料分類（Data/）

LD3 資料分 9 大類，跟 LD4 結構一致（系列共通設計）：

  • Map 97 MB — 地圖資料（最大）
  • Monster 42 MB — 怪物 sprite（用 .TIL 格式，每隻怪物有多個動作檔）
  • Character 24 MB — 角色 sprite
  • Effect 4.4 MB — 戰鬥特效
  • Weapon / Shield 約 3 MB — 武器 / 盾牌資料
  • Campaign / Range / MSG — 劇本 / 範圍 / 訊息字典

⚠️ 怪物 / 角色 sprite 是創作性表現（屬版權保護內容），本站不公開。

.rsrc 內含 UI 字串（2026 提取）

從 LD3.exe .rsrc resource section 提取出 389 個唯一中文 UI 字串（UTF-16LE 編碼，跟 LD4 共用 300 個）。揭露的核心系統設計：

  • 連線設計：「次元之門設定」/「次元之門的狀態」/「確認性開放」/「限制性開放」/「LAN 內自動檢索」/「選擇伺服器」/「第三波伺服器」/「接受上限人數(包含其他 PC 隊員)」
  • 世界系統：「世界簡介」/「使用者代號」/「選擇要前往的世界」/「劇本編輯」
  • 玩家屬性：「體力」/「魔力」/「屬性 1-2」/「信仰屬性」/「年齡」/「性別」/「稱號」
  • 遊戲系統：「武器資料編輯」/「對象物品設定」/「報酬設定」/「場所設定」/「狀態設定」

→ 確認 LD3 也有「次元之門」連線系統（不只是 LD4 才有）— 系列共通設計。

res/MonsterPara.BIN — 146 怪物名（明文！）

LD3 主資料 res/MonsterPara.BIN（31,788 bytes，\\0LD3 magic header）沒有加密，可直接逆向。提取出 146 隻怪物名，揭露俠客遊系列的設計傳承：

  • 共通最終 boss：「安哥摩爾」(× 4 變體) — 跟俠客遊 II的「魔王安哥爾莫亞」同一個！系列共通的最終魔王
  • 4 元素精靈（× 4 變體）：火 / 水 / 風 / 土 — 跟 II 完全一致
  • 4 狂信者：光 / 闇 / 混沌 / 秩序 — 對應第三之書的 4 卡片陣營屬性，從 LD3 (2000) 起就存在
  • 4 變體系統：許多怪物名重複 4 次（撒旦 × 4 / 天馬 × 4 / 九尾狐 × 4 等）— 暗示有「弱 / 中 / 強 / 王」4 級 sub-id
  • 不死系：骷髏 / 狂骨人 / 骸骨人 / 殭屍 / 死屍 / 惡鬼 / 腐屍妖
  • 4 種天使：秩序 / 混沌 / 光 / 闇 各 4 變體 = 共 16 種天使

res/ 完整檔案統計（36 BIN + 187 BMH + 24 TIL）

  • 36 個 .BIN：全部 \\0LD3 magic（明文，無加密）
  • 187 個 .BMH：BitmapHeader 圖像 — magic 4 bytes 其實是 file size（5 個主要 size 範圍）
  • 24 個 .TIL：tile 圖像，magic 20 04 00 00 = 1056（推測 width × height）

★ res/ 各 mt_*.BIN — 1,183 個明文中文字串

LD3 res/ 各 mt_*.BIN 跟 MonsterPara.BIN 全部明文，提取出 1,183 個 unique 中文字串，這是 LD3 隨機世界生成器用的 name pool：

  • 世界 / 國家名 516 個（mt_world.BIN）：阿修連 / 阿古納拉達 / 阿古雷特 / 阿茲里亞 / 阿托地克 等
  • 城鎮名 144 個（mt_f_town.BIN）：開始之鎮 / 冒險者之鎮 / 月之鎮 / 博愛之鎮 等
  • 怪物名 87 個（MonsterPara.BIN）：食人觸手 / 鬼魂 / 安哥摩爾 / 4 元素精靈 等
  • 自然之地名 89 個（mt_f_nature.BIN）：繁華之地 / 背叛之地 / 神秘之地 等
  • 人類迷宮名 87 個（mt_d_human.BIN）：斷絕的迷宮 / 虛無的迷宮 / 疑惑的迷宮 等
  • 自然洞窟名 78 個（mt_d_nature.BIN）：偽之洞窟 / 謊言洞窟 / 虛偽的洞窟 等
  • 連結門 / 祠名 70 個（mt_f_linkgate.BIN）：神聖之祠 / 神秘之祠 / 狂信之祠 等
  • 地形名 14 個（mt_element.BIN）：灼熱地帶 / 暖濕地帶 / 砂漠地帶 等

→ 每次新遊戲，LD3 系統從這些 pool 隨機選名字組合產生獨一無二的世界。LD4 進一步把這設計擴充成 Web/Models 模板，可以分享給其他玩家。
