[ 光碟結構分析（2026 逆向）]

2026 年 toni 將實體光碟拷貝到電腦後，本站對 LD3 完整光碟結構做了二進位分析：

主程式

  • LD3.exe — 2,494,898 bytes（約 2.4 MB），編譯日期 2000-11-17，PE32 格式（Windows 32-bit i386 GUI）
  • 跟第三之書（2002）一樣是 MFC 程式，但 LD3 比第三之書早 2 年發行

光碟結構

  • 安裝光碟（LD3_INSTALL）：DirectX 安裝程式 + DirectXMedia + 主程式（ld3 資料夾）
  • 遊戲光碟（LD3_GAME_DISC）：Movie 動畫 + initial
  • 遊戲資料總大小約 275 MB

遊戲資料分類（Data/）

LD3 資料分 9 大類，跟 LD4 結構一致（系列共通設計）：

  • Map 97 MB — 地圖資料（最大）
  • Monster 42 MB — 怪物 sprite（用 .TIL 格式，每隻怪物有多個動作檔）
  • Character 24 MB — 角色 sprite
  • Effect 4.4 MB — 戰鬥特效
  • Weapon / Shield 約 3 MB — 武器 / 盾牌資料
  • Campaign / Range / MSG — 劇本 / 範圍 / 訊息字典

⚠️ 怪物 / 角色 sprite 是創作性表現（屬版權保護內容），本站不公開。

.rsrc 內含 UI 字串（2026 提取）

從 LD3.exe .rsrc resource section 提取出 389 個唯一中文 UI 字串（UTF-16LE 編碼，跟 LD4 共用 300 個）。揭露的核心系統設計：

  • 連線設計：「次元之門設定」/「次元之門的狀態」/「確認性開放」/「限制性開放」/「LAN 內自動檢索」/「選擇伺服器」/「第三波伺服器」/「接受上限人數(包含其他 PC 隊員)」
  • 世界系統：「世界簡介」/「使用者代號」/「選擇要前往的世界」/「劇本編輯」
  • 玩家屬性：「體力」/「魔力」/「屬性 1-2」/「信仰屬性」/「年齡」/「性別」/「稱號」
  • 遊戲系統：「武器資料編輯」/「對象物品設定」/「報酬設定」/「場所設定」/「狀態設定」

→ 確認 LD3 也有「次元之門」連線系統（不只是 LD4 才有）— 系列共通設計。
