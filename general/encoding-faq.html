[ 亂碼及字型問題 ]

基本上通用解決亂碼辦法 : 使用翻譯軟體( 例如 : 譯典通 ) 開啟多語言瀏覽。

WINDOWS 95/98/98SE解決方法:

第一種：直接從[控制台/新增移除]裡把IE 5.0的日文字形給移除掉就行了，移除時請選擇"Japanese Language Support"然後選移除再重新開機就行了，使用此種方式是可以一勞永逸，以後玩俠客遊都不會有亂碼，但是對於有在用中文的IE5.0來上日文網頁或是有在用CWIN在玩JWIN遊戲的人卻會發現字都會看不到。

第二種：因為在有裝日文字形的情況之下，Windows的資料夾底下的Fonts裡面，應該會有"MS Gothic"(IE 5.0)或"MS Mincho"(Office)這兩個字形(在某些情況之下只有一個，不影響)，這兩個檔案就是日文字形的字形檔，只要在玩俠客遊時將這兩個檔案暫時移動 到別的資料夾就行了，等到要瀏覽日文網頁或是要玩JWIN遊戲時在將其移動回來就行了^^，好處是不用把日文字形砍掉，缺點就是檔案要動來動去，滿麻煩 的。

(推薦使用方法)第三種：

（ＸＰ執行方法 : 按左下角的執行 -> 開始 -> 輸入Regedit -> 按確定)

找到Windows資料夾底下的Regedit.exe檔，然後在裡頭找到以下的位置

[HKEY_LOCAL_MACHINE\System\CurrentControlSet\control\fontassoc\Associated CharSet]

然後可以看到以下字串:

"ANSI(00)"="yes"

"SYMBOL(02)"="no"

"OEM(FF)"="yes"

"CHINESBIG5(88)"="yes"

增加一組【字串值】...如下

"SHIFTJIS(80)"="yes"

建議先備份regedit整個登錄檔，

改了之後，重開機，再執行遊戲就沒亂碼了!

WINDOWS ME 解決亂碼方法:

安裝南極星軟體即可。

未來之書 For WINDOWS 2000 & XP 解決亂碼方法：

作者：李逍遙

前途道標 For WINDOWS XP 字型問題解決 :

GameBase 作者： singlau (孤星劍)

在XP的正常情況下，與同伴或路人交談時， 出現的小方塊會產生字體重疊，令對話看不清，

解決方法是, 將顯示卡的字型大小, 由 "正常大小" (96DPI) ，轉為 "大" (120DPI)

開始 -> 控制台 -> 顯示 -> 外觀 -> 字型大小 -> 改大字 -> 確定 -> 重開機

前途道標 For Windows 10 遊戲內字體會重疊在一起問題解決:(New 2019/12/11更新)

toni在網路上爬文，發現DPI要調成4的倍數的資訊，得到網路上的一個數值調整:124%，

設定成這個值後，就可以看清楚遊戲內的字體了。

一、先到遊戲資料夾，在遊戲執行檔LD95.exe上按右鍵，把內容設定成以下圖片一模一樣，

接著點「變更高DPI設定」。

二、接著一樣設定成跟圖片一模一樣，之後點「開啟進階縮放設定」。

三、再來也是設定跟圖片一模一樣，之後Windows會請你先登出後再登入即可。

修改完成後的遊戲內字體畫面，就至少是可以看得懂的狀態了^_^

俠客遊IV字體過小解決辦法 :

巴哈姆特 作者 : hcmhcm

請編輯 遊戲資料夾下的 ld4.ini

把 MessageLogDlg_Bottom= 的數值多加 15 就可以了。
