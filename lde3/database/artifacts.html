第三之書 神器圖鑑（71 件，11 大類）

⚠️ 資料來源說明（2026 跨檔比對確認）：跨 7 個 EXE 變體檢查後確認，這 71 件神器中：
  • 2 件（祭品之杖 / 單手戰斧）出現在 Lde3.exe v1.2 patch — 直接從遊戲主程式提取的事實
  • 2 件（布吉之斧 / 巴克拉之盾）對應到 Lde3.exe items array 的 ID 248 / 256
  • 67 件只出現在第三之書精靈工具 Lde3ET的內建表，不在任何版本 Lde3.exe 主程式
換句話說，大多數神器名稱來自當年社群製作的精靈工具的內建資料表 — **可能是玩家實際遊戲中遇到的真實神器**（精靈工具是給玩家修改角色裝備用的，名單應該對應實際遊戲），但**未從遊戲主程式二進位直接驗證**。

內容：第三之書神器級武器命名橫跨多文化神話系統，本頁列出 71 件神器依文化系統分類。

命名來源：
  • 遊戲原創套裝：光榮系列 11 件 + 混沌系列 4 件 + 暗黑系列 4 件
  • 中華神話：七星劍 / 九龍槍 / 玉璽 / 青龍刀（關公青龍偃月刀典）等
  • 希臘神話：大阿波羅之弓 (Apollo) / 阿爾提蜜絲之弓 (Artemis)
  • 北歐神話：巴爾基里之槍 (Valkyrie 女武神)
  • 印度 / 佛教：阿修羅之弓 / 夜叉王之劍
  • 元素 / 自然：火炎 / 火燄 / 大地 / 地脈 / 星屑 / 時空
  • v1.2 修正：祭品之杖（v1.0 原譯遠古之杖）

合規說明：神器名稱屬遊戲事實層內容（玩家於商店、寶箱、戰利品即可看見），依日本著作權法第 2 條僅創作性表現受保護，事實名稱與其文化典故來源不在保護範圍。
