第三之書 LDE3.EXE 逆向字串字典（810 個有效中文字串，第三十九波 toni 提取）

方法：對 Lde3.exe (PE32, 2.5MB) 用 Big5 模式（high byte 0xA1-0xC6）+ ≥4 中文字 + null terminator 過濾，去重後分類。

合規：純名字 / 系統字串字典（事實層，依日本著作權法第 12 條²不受資料庫著作權保護），不含創作性編排。

用途：玩家遇到不熟字串時可在此查詢含義 / 玩家社群核對遊戲系統翻譯。完整 JSON 在 strings.json。
