第三之書瀏覽器版修改器（v0.1）— 跨平台 / 純前端 / 無上傳。

支援檔案：lde3s.le3（主存檔，985,542 bytes）位於遊戲資料夾/usr/。
已知 header：byte 0-3 = "le3\0" magic / byte 4-31 = "LUNATIC DAWN - The 3rd Book -" + nulls / byte 32-63 = "Copyright (C) ARTDINK Corp. 2000" + nulls / byte 0x54+ = binary game data。

⚠️ 修改前務必備份：
1️⃣ 先把整個 usr/ 資料夾複製為 usr_backup/
2️⃣ 在備份上測試 + 載入遊戲驗證
3️⃣ 確認 OK 後再覆蓋原檔
（吸收 RedTime「寫工具與工具不同步」教訓）

本工具當前功能：
▸ 拖放 lde3s.le3 → header 驗證 + 結構區段識別
▸ Hex viewer（前 1024 bytes）
▸ Hex 編輯器（任意 byte 直編）
▸ 下載修改後 SAV

未來版本（v0.2+）：field-level 結構解析（金錢 / 等級 / 屬性 / 物品），需逆向 lde3s.le3 結構。

所有解析在瀏覽器本地完成，不上傳任何資料。
