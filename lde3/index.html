Lunatic Dawn: The Third Book (2002) · 俠客遊 Book 三部曲第三作 · 台灣美商藝電代理

第三之書（Lunatic Dawn: The Third Book，簡稱 LDE3）是 2002 年由台灣美商藝電代理發行的俠客遊系列作品，是繼 1998 年未來之書（The Book of Futures）、1999 年前途道標（Passage of The Book）之後的「Book 三部曲」第三作。執行檔版本號 1.0.8.2（官方台灣繁體中文版 1.2 更新檔）。

本站收錄歷年來社群製作的工具與修正檔——包含 Morrowind 製作的 ELF 工具、kkarthur 的修改器、askacheng 的管理員、anDY & aSKa STUDiO 的流動化啟動器，以及解決 Win98 / XP / Win11 各時代相容性問題的修正檔。

[ INSTALL · 安裝設定 SOP ]

第三之書原版光碟跨 Win98 / XP / Vista / 7 / 8 / 10 / 11 不同時代有不同安裝障礙。以下是 toni 跨年代維護整理的標準流程：

  1. 確認執行檔版本為 1.0.8.2——這是台灣繁體中文版官方 1.2 更新檔的版本號。版本不對某些工具會無法正確 hook。
  2. 遊戲資料夾路徑用簡潔英文，避免中文字 / 空格 / 過深層級。建議：C:\ARTDINK\Lde3\ 或可攜版直接放隨身碟根目錄。
  3. 套官方 1.2 更新檔——把官方修正檔解壓後複製到遊戲資料夾覆蓋。
  4. NoCD 化（避免每次都要放光碟）——第三之書保護光碟檔解壓後跑，會建立 NoCD 環境。
  5. 解 Win11 / XP / Win98 相容性（如果遇到「試玩版」或「找不到光碟」問題）——下載對應系統的登錄檔修正檔解壓後執行。
  6. 輔助工具放遊戲資料夾內執行（重要！相對路徑解析）——修改器 / ELF 工具 / 日期計算器都直接放在跟 LDE3.EXE 同層目錄。
  7. 登錄檔路徑用雙斜線——若手動編輯登錄檔，路徑必須寫雙斜線，例：
    c:\\Program Files\\ARTDINK\\Lde3\\LDE3.EXE

[ FAQ · 常見問題排除 ]

Q1. 安裝完跑遊戲變成「試玩版」？

原因：登錄檔內遊戲執行檔路徑沒寫對 / 系統升級後路徑失效。
解法：依作業系統下載對應修正檔——lde3reg.zip（通用）/ XP 版 / Win11 版。解壓後執行裡面的 .reg 或 .bat 檔即可。

Q2. 開啟遊戲提示「找不到光碟」？

原因：未套用 NoCD 修正 / 光碟認證機制在新作業系統上失效。
解法：套第三之書保護光碟檔（最新版）或歷史 NoCD 版。

Q3. 想直接攜帶玩不要安裝？

解法：用PortableLde3_tw1082r1.zip 可攜版，解壓即用。適合放隨身碟 / SSD 帶到不同電腦玩。

Q4. 修改器抓不到存檔 / 數值改完沒反應？

原因：執行檔版本不對 / 修改器版本舊 / 修改器跟 LDE3.EXE 不在同層目錄。
解法：確認執行檔 1.0.8.2 + 用最新第三之書修改器 v0.4（kkarthur 製作）+ 放在跟 LDE3.EXE 同資料夾。

Q5. 想看原版手冊 / 認證碼說明？

歷史第三之書管理員 1.03 / 1.06 包含部分手冊掃描跟說明。最新版用第三之書管理員 1.0.6（askacheng 製作）。

Q6. 為什麼推薦的工具製作年份是 2007-2010？

第三之書 2002 年發行後，2007-2010 年是社群最活躍時期，多數修正工具完成於該段時間。Morrowind / kkarthur / askacheng 等社群開發者貢獻的工具至今仍是各時代最佳維護方案。
