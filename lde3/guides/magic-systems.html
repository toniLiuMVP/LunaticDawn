[ 5 系魔法概觀 ]

第三之書沿用 Book 三部曲 5 系魔法系統，跟俠客遊 II 共用大多數魔法。

魔法系學習場所特性
物理系魔法屋 / 訓練所輔助 / 控制（照明 / 開鎖 / 索敵 / 護身）
元素系魔法屋 / 訓練所攻擊性（爆炸 / 波彈 / 隕石 / 暴風雨）
神聖系僅神聖系寺院治療 / 復活 / 對暗黑類 boss
暗黑系僅暗黑系寺院詛咒 / 召喚 / 對神聖類 boss
精靈系精靈術屋（馬尼卡帕王國）自然系 + 4 召喚獸（伊弗利朵 / 琴恩 / 克拉肯 / 貝希摩斯）

[ 玩家路線建議 ]

  • 物理 / 元素：通用路線，初期就學
  • 神聖 vs 暗黑：選一條走（衝突）— 神聖適合「正義路線」，暗黑適合「邪惡路線」
  • 精靈系：後期接觸（需到馬尼卡帕王國），4 召喚獸非常強力
