[ 15 種任務類型（從 LUNA2.EXE 提取）]

  • 送貨 / 購物 / 護衛 / 探索 / 修行
  • 復甦 / 決鬥 / 調查 / 擊退 / 奪回
  • 救出 / 強盜 / 暗殺 / 綁架 / 收買

[ 任務難度分級 ]

  • 低風險（新手）：送貨 / 購物 / 護衛
  • 中風險：探索 / 修行 / 擊退 / 救出
  • 高風險（進階）：決鬥 / 強盜 / 暗殺 / 奪回
  • 道德爭議（邪惡路線）：強盜 / 暗殺 / 綁架 / 收買

[ 任務經濟學 ]

  • 送貨 / 購物：穩定但收入低（適合每日小額）
  • 護衛 / 救出：中等收入，須準備戰鬥
  • 決鬥 / 暗殺：高報酬高風險，後期主力
  • 修行：經驗值優先，金錢次要

[ 委託期限機制 ]

每個任務有期限（遊戲日數），逾期任務失敗。建議：

  • 不接超出當前實力的任務
  • 不同時接太多任務（避免時間衝突）
  • 用第三之書日期計算器（Morrowind）規劃時程
