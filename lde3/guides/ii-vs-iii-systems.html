[ 卡片戰鬥的隱藏系統 ]

從 CoCard.dll（卡片戰鬥引擎 DLL，352 KB）逆向取得：

  • 4 種卡片陣營屬性：黑暗 / 神聖 / 混沌 / 秩序 — 每張卡片屬於一個陣營
  • 支援卡片 vs 攻擊卡片：卡片戰鬥區分支援型與攻擊型
  • 世界分享下載功能（v1.0）：玩家可把自己的世界檔上傳到伺服器，分享給其他玩家下載 — 完整 TCP/IP 通訊機制（主機解決 / 受信 / 送信 / 接續 等）支撐這個分享系統

系列功能對照：第三之書是「世界分享下載」（非即時對戰）。即時網路對戰要到俠客遊 III 跟俠客遊 IV才有。第三之書的世界分享伺服器多年前已關閉，但 DLL 與 EXE 內仍保留完整通訊邏輯，這是第三之書設計史上重要的歷史事實。

結論：第三之書並非「俠客遊 II 加強版」，而是同世界觀 + 同任務概念上的戰鬥系統大改寫。II 的老玩家轉到第三之書時，最大學習曲線在卡片戰鬥的牌組構築。

選擇建議：偏好傳統 RPG 玩法 → II；偏好策略 / 牌組構築 → III。

版本選擇：建議套用 v1.2 patch，譯名跟 II 一致 + 字串修正。
合規說明：本對照表所列差異全屬玩家於遊戲中可見之事實（屬性名稱 / 任務類型名稱 / 系統機制存在與否 / patch 修正紀錄），依日本著作權法第 2 條僅創作性表現受保護，事實名稱與系統觀察不在保護範圍。
