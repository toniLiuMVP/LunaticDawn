[ 戰鬥指令系統 7 種 ]

第三之書沿用 Book 三部曲戰鬥指令架構：

  • 喊叫：向部隊全員打招呼，發出戰鬥指令
  • 攻擊敵人：對指定敵人 / 全體攻擊
  • 使用道具：用攜帶物品（指定個人，不對全體）
  • 解救：援助受困同伴
  • 治療：用魔法治療單一同伴
  • 跳躍：戰鬥撤退（指示全員時自己也撤）
  • 使用魔法：5 系魔法施展

[ 7 種弱點屬性 ]

從 LUNA2.EXE 字串提取確認 7 種弱點：

  • 沒有：無弱點
  • 物理：對物理攻擊弱
  • 炎熱：對火系魔法弱
  • 冷卻：對水系魔法弱
  • 電光：對雷系魔法弱
  • 神聖：對神聖系魔法弱（不死類專弱點）
  • 暗黑：對暗黑系魔法弱（神聖類專弱點）
  • 精靈：對精靈系魔法弱

查詢怪物弱點：遊戲資料庫 · 怪物頁（俠客遊 II 數據，多數怪物概念延續到第三之書）。

[ 攻擊類型對應 ]

  • 砍：刀劍類武器
  • 刺：槍類 / 雙手槍
  • 敲：棍棒類 / 拳

怪物對不同攻擊類型有不同抗性。例：骷髏類對「砍」抗，對「敲」弱。

[ Boss 戰術 ]

  • 確認弱點：戰前用「探查術」（物理系）確認 boss 弱點
  • 準備對應魔法：依 boss 弱點屬性帶 ≥ 3 個施法者
  • 輔助先行：護身術 + 索敵術 + 沉默術（避免 boss 施法）
  • 必殺技時機：留高 STM 等 boss HP < 30% 時施展
  • 治療配置：至少 1 個僧侶 / 神聖系魔法 NPC 同行
