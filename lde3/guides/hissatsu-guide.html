[ 8 大武器類必殺技 ]

從 LUNA2.EXE binary 逆向確認的 8 個必殺技（沿用 Book 三部曲架構）：

  • 拔刀波斬流（劍）— 主形 STM 60 / 變體 110
  • 暴棍撲術（棍棒 1）— 主形 STM 85 / 變體 115
  • 直棍撲術（棍棒 2）— 主形 STM 88 / 變體 110
  • 真槍貫術（槍）— 主形 STM 92 / 變體 118
  • 爆矢眼弓法（弓）— 主形 STM 84 / 變體 108
  • 剛投爆石（投擲）— 主形 STM 87 / 變體 104
  • 波動六骸法（拳）— 主形 STM 60 / 變體 110
  • 斷強破防（內功）— 主形 STM 70 / 變體 95

[ 使用時機 ]

  • 對 boss 留 STM：boss HP < 30% 時施展變體（高威力）
  • 連戰要省 STM：群戰用主形 / boss 戰用變體
  • 進攻路線：劍 / 棍棒 / 槍 / 弓玩家選 1-2 武器類專精
