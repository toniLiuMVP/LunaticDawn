[ 第一步：環境準備 ]

第三之書是 Win 原生程式（不需 DOSBox），但 1996 年代的程式對 Win10 / Win11 有相容性問題。

  1. 安裝路徑：英文簡潔，例 C:\ARTDINK\Lde3\（避免中文 / 空格 / 過深）
  2. 套官方 1.2 更新檔：執行檔升到版本 1.0.8.2
  3. NoCD 化：套保護光碟檔，避免每次都要放光碟
  4. 登錄檔修正：Win11 套LDE3win11.7z，XP 套Lde3XP.zip

完整安裝 SOP 見：▸ 安裝設定 SOP + FAQ

[ 第二步：角色設定 ]

遊戲開始進入「角色設定」需要 6 個項目：

  • 姓名：自由輸入（中文 1-3 字）
  • 稱號：可在遊戲中變更，影響 NPC 反應
  • 性別：開局決定不能變（含雙性人第三性別）
  • 家系：影響起始金錢 / 屬性（貴族 156 vs 孤兒 130 屬性點 / 10000 vs 3000G）
  • 種族：人類 / 小妖精 / 侏儒 / 半精靈 / 半小妖精
  • 職業：戰士 / 魔法師 / 盜賊 / 僧侶（可雙職業組合）

[ 跟前作的差異 ]

沿襲前作的設計

  • 開放世界 RPG 結構（無強制主線）
  • 城鎮 + 地下城 + 戰鬥地圖三段式
  • 5 系魔法 + 8 屬性 + 15 技能熟練度
  • 同行 NPC 系統 + 子分 / 親分

第三之書新增 / 改良

  • Win 原生引擎（不需 DOSBox）
  • 美術 / 圖像更精緻
  • 新地圖 + 新城鎮
  • 更精緻的 NPC 互動
  • 豐富的修正工具支援（社群 2007-2010 釋出）

[ 第一週玩法建議 ]

  • Day 1-2：到旅店看委託，選「送貨 / 護衛」低風險任務累積資金
  • Day 3-7：到訓練所提升核心技能熟練度（依職業選 1-2 個重點技能）
  • Week 2：招募有互補技能的同行 NPC，準備地下城探險
  • Week 3+：準備充足燈油（建議 50 小時+）+ 鑰匙（或盜賊 NPC），開始挑戰地下城
