[ 5 階段怪物分類 ]

第三之書沿用 Book 三部曲 128 怪物 architecture（MONSTER.DAT 結構共用）：

  • 新手期 (Lv 3-7)：盜賊 / 山賊 / 野狼 / 蜘蛛 — 練功用
  • 中期 (Lv 10-20)：戰士 / 騎士 / 武士 / 妖魔 — 累積金錢裝備
  • 中後期 (Lv 25-35)：龍類 / 巨人 / 高位魔法師
  • 後期 (Lv 40-49)：不死類 / 異種 / 龍王
  • 終局 (Lv 50)：6 隻 boss（魔王阿迦魯瑪 HP 999 等）

[ 通用 boss 戰術 ]

  • 戰前用「探查術」確認 boss 弱點
  • 對應魔法系玩家放在前排
  • 輔助先行：沉默 + 護身
  • 必殺技留到 boss HP < 30% 施展
