[ 安全修改 SOP ]

⚠️ 修改前務必備份

  1. 先確認執行檔版本為 1.0.8.2（執行檔升不對，修改器抓不到存檔）
  2. 備份原檔：把整個 USER0/（或對應存檔資料夾）複製為 USER0_backup/
  3. 修改器放遊戲資料夾：跟 LDE3.EXE 同層
  4. 先小幅修改測試：例如金錢 +1000，看遊戲讀取是否正常
  5. 確認無誤再大幅修改：屬性 / 物品 / 等級
  6. 修改後 save / quit / re-load 驗證：避免存檔損毀

[ 第三之書修改器 v0.4 (kkarthur) ]

下載：Google Drive 主連結 · 備用（密碼：lunaticdawn）

基本用法

  1. 把 Lde3Edit_v0.4.exe 放在跟 LDE3.EXE 同資料夾
  2. 遊戲先 save 一次（產生最新存檔）
  3. 關掉遊戲
  4. 跑 Lde3Edit_v0.4.exe
  5. 選擇要修改的角色 + 屬性
  6. 修改完按「儲存」
  7. 重新進遊戲讀取存檔，確認修改生效

可修改項目

  • 角色屬性（力 / 智 / 靈 / 敏 / 直 / 魅 / 體 / 精）
  • HP / MP / STM 上限
  • 金錢
  • 物品欄
  • 5 系魔法學會狀態
  • 技能熟練度

[ ELF 工具 v1.62 (Morrowind) ]

下載：Google Drive

ELF 工具編輯遊戲資料的 ELF 加密檔。比修改器更深入，可改：

  • 遊戲內物品定義（價格 / 攻擊力 / 防禦值）
  • 怪物數值
  • 地圖事件參數
  • NPC 設定

⚠️ ELF 工具修改的是遊戲資料檔（不是存檔），所有玩家都會受影響。建議：

  • 先備份整個遊戲資料夾
  • 了解 ELF 結構再修改（Morrowind 工具有說明文件）
  • 適合「想改造遊戲體驗」的進階玩家

[ 修改器使用建議 ]

  • 第一輪正常玩：體驗成長感跟挑戰
  • 卡關才用修改器：避免一開始就破壞樂趣
  • 修改幅度漸進：先小修，看遊戲反應，再大修
  • 用 USER1 測試 USER0 保留：作為對照
  • 修改後保留多個存檔點：避免單點故障
