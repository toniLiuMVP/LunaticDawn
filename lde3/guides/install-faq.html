[ 安裝 SOP · 7 步驟 ]

⭐ 跨 Win98 / XP / Vista / 7 / 8 / 10 / 11 不同時代的標準流程：

  1. 確認執行檔版本為 1.0.8.2（台灣繁中版官方 1.2 更新檔）
  2. 遊戲資料夾路徑簡潔英文：C:\ARTDINK\Lde3\（避免中文 / 空格 / 過深層級）
  3. 套官方 1.2 更新檔：解壓官方修正檔覆蓋遊戲資料夾
  4. NoCD 化：套第三之書保護光碟檔
  5. 解作業系統相容性：對應系統下載修正檔（通用 / XP / Win11）
  6. 輔助工具放遊戲資料夾內（相對路徑解析）
  7. 登錄檔路徑用雙斜線：c:\\Program Files\\ARTDINK\\Lde3\\LDE3.EXE

[ FAQ · 6 個常見問題 ]

Q1. 安裝完跑遊戲變成「試玩版」？

原因：登錄檔內遊戲執行檔路徑沒寫對 / 系統升級後路徑失效。
解法：依作業系統下載對應修正檔執行。

Q2. 開啟遊戲提示「找不到光碟」？

原因：未套 NoCD 修正 / 光碟認證機制在新系統失效。
解法：套保護光碟檔。

Q3. 想直接攜帶玩不要安裝？

解法：用PortableLde3_tw1082r1.zip 可攜版。

Q4. 修改器抓不到存檔 / 數值改完沒反應？

原因：執行檔版本不對 / 修改器版本舊 / 修改器跟 LDE3.EXE 不在同層目錄。
解法：確認執行檔 1.0.8.2 + 用最新修改器 v0.4 + 放在跟 LDE3.EXE 同資料夾。

Q5. 想看原版手冊 / 認證碼？

歷史第三之書管理員 1.03 / 1.06 包含部分手冊掃描。

Q6. 為什麼推薦工具是 2007-2010 年的？

2007-2010 是社群最活躍時期，多數修正工具完成於該段時間。Morrowind / kkarthur / askacheng / anDY & aSKa STUDiO 等社群開發者貢獻的工具至今仍是各時代最佳維護方案。
