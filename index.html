[ ABOUT ]

俠客遊 (Lunatic Dawn) 是由日本 Artdink 製作、台灣第三波/美商藝電代理的開放世界 RPG。

這個網站保存了 2000 年 GameBase 遊戲基地、巴哈姆特俠客遊討論板 社群前輩整理的攻略、物品、存檔格式文件，並用現代瀏覽器技術重新包裝——讓這些 快要被遺忘的資料在 2026 年後，仍然能被後世玩家取用。

※ 本站基於數位文化保存目的整理，遊戲本體著作權屬於 Artdink。站上技術文件為獨立逆向研究成果，遊戲內美術 / 劇情文字 / 配樂等創作性內容僅作為研究者個人參考保留，不在本站公開展示。 請至 Steam 購買正版 俠客遊 I/II/III 合輯或前途道標支持原作。若您是攻略原作者並希望授權變更或撤除，請至 GitHub Issues 聯繫。

⭐ 2026-05-13 · toni 版 Luna2 懶人包升級為「有音樂音效版」 · 本專案起源

過去 20+ 年網路流傳的 toni 版 Luna2 是「無音樂音效版」(早年 XP 時代音效驅動不易處理)。2026 年透過 Claude Code 技術升級,正式進化為:

  • ✅ 有音樂音效版 — DOSBox Staging SoundBlaster 模擬,1996 原版 BGM 與音效全部還原
  • ✅ 繁體中文化(1996 第三波代理版)
  • ✅ 跨平台 — macOS / Windows / Linux 三平台啟動腳本
  • ✅ 解壓即玩 — 雙擊啟動,不用裝 DOSBox / 改設定 / 處理音效驅動

玩家流程(2 種選擇 → 都得到中文化 + 有音樂音效 + 跨平台):

  • 🅰️ Steam 玩家:先至 Steam 購買 俠客遊 I/II/III 合輯(僅日文介面)→ 下載懶人包 → 用 Steam 版遊戲原檔覆蓋 → 雙擊啟動
  • 🅱️ 1996 老磁碟片玩家:把你 30 年前的正版實體磁碟片(1.44 MB × 5 片)安裝後的 LUNA2.EXE + 資料檔丟進懶人包 game/ → 雙擊啟動

▌ 站長傳承

「吟遊詩人的傳說 · 俠客遊小站」由 1999 年 BBS 時代的兩位創站站長 小傑(LemiyaJay)與 小蜜蜂(Yellowbee)創立。21 世紀初(2000 年代初期),toni 從小傑與小蜜蜂手上接手俠客遊小站,持續維護至今 — 後續歷經 Google Pages(2009)→ Google Sites(2010 年代)→ GitHub Pages(2026)三次平台遷移,但站名與精神不變,延續這份俠客遊系列的數位文化傳承。

感謝小傑與小蜜蜂兩位創站站長,沒有他們 1999 年的 BBS 時代起點,就沒有 27 年後本站的存在。
※ 「吟遊詩人的傳說 · 俠客遊小站」站名為 1999 年小傑 / 小蜜蜂創立時自然形成的通稱,toni 接手後持續使用此名延續歷史傳承。本站非任何官方授權,純玩家社群文化保存。
